[Setup]: ArchitecturesInstallIn64BitMode

Возможные значения:
Одно или больше из следующих значений, разделенных пробелами:
x64
ia64

Значение по умолчанию:
(пустой)
Описание:

Определяет 64-разрядную архитектуру(ы) процессора, на которой инсталлятор будет работать в 64-разрядном режиме. Если директива не определена или пустая, то инсталлятор будет всегда работать в 32-разрядном режиме.

Обычно, не нужно изменить значение по умолчанию, если Ваше приложение не содержит 64-разрядные двоичные файлы.

Убедитесь, что Вы читали тему ограничения установки в 32-разрядном режиме прежде, чем использовать эту директиву.

Если Ваше приложение работает только на 64-разрядной архитектуре процессора, следует установить ArchitecturesAllowed в значение данной директивы, чтобы препятствовать установке программы на 32-разрядный Windows.

Инсталлятор может работать только в тех 64-разрядных версиях Windows, которые имеют поддержку API, требуемый для работы инсталлятора (например. RegDeleteKeyEx). Все x64 выпуски обеспечивают необходимую поддержку API, однако версии Itanium до Windows Server 2003 SP1 нет. Если пользователь имеет более старую версию Itanium Windows, то инсталлятор покажет сообщение (MissingWOW64APIs) рекомендуя пользователю установить пакет обновления и завершится.
