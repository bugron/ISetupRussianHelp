[Setup]: AllowNoIcons

Возможные значения:
yes or no

Значение по умолчанию:
no
Описание:

При значении yes, на странице Выбор папки меню Пуск инсталлятор отобразит флажок Не создавать папку в меню «Пуск», который позволяет пользователю не создавать ярлыки в меню «Пуск».

По умолчанию влияет только на те записи секции [Icons], у корторых параметр Name начинается с {group}\ и у которых нет параметра Tasks. Для других записей можно добавить параметр Check: not WizardNoIcons.
