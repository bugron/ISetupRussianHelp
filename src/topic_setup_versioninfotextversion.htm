[Setup]: VersionInfoTextVersion

Значение по умолчанию:
VersionInfoVersion
Описание:

Определяет текстовый файл со значением версии для информации о версии Установки.

Имейте ввиду, что это значение показывался только на вкладке Version проводника в Windows 98 и ранее. Более поздние версии показывают двоичное значение версии (VersionInfoVersion).

Данная директива игнорируется, если UseSetupLdr=no.
