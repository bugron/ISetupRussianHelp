[Setup]: UninstallLogMode

Возможные значения:
append, new, или overwrite

Значение по умолчанию:
append
Описание:

При значении append деинсталлятор по возможности дополняется.

При значении new создается новый деинсталлятор.

При значении overwrite существующий деинсталлятор того же приложения перезаписывается, а не дополняется (использование не рекомендуется). Для дополнения деинсталлятора действуют те же правила, что и для перезаписывания.

Например:
UninstallLogMode=append
