Pascal Scripting: StrToInt64Def

Прототип:

function StrToInt64Def(s: string; def: Int64): Int64;

Описание:

Функция конвертирует строку, переданную через переменную S, в 64-битное число. Если результат конвертации не представляет собой допустимое 64-битное число, функция возвращает 64-битное число Def.

Требуется Unicode Inno Setup.
