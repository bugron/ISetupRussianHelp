Pascal Scripting Extended: NewPopupMenu

Прототип:

function NewPopupMenu(Owner: TComponent; const AName: String; Alignment: TPopupAlignment; AutoPopup: Boolean; Items: array of TMenuItem): TPopupMenu;

Описание:
Функция создает, во время выполнения программы, и инициализирует всплывающее меню, элементы которого заданы параметром Items.
Параметр Owner определяет компонент-владелец, с которым будет ассоциировано меню. Параметр AutoPopup определяет, должно ли появляться меню при щелчке правой кнопкой мыши на компоненте. В параметре AName указывается имя, которое будет идентифицировать всплывающего меню в исходном коде программы. Параметр Alignment определяет, в каком месте относительно указателя мыши появится меню.
Элементы меню можно создать с помощью функции NewItem.
Функция NewPopupMenu не может ассоциировать всплывающее меню с элементами управления.
