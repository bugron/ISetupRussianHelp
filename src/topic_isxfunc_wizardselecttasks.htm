Pascal Scripting: WizardSelectTasks

Прототип:

procedure WizardSelectTasks(const Tasks: String);

Описание:

Выбирает указанный список задач, разделенных запятыми.

Если имя задачи начинается с символа "!", то выбор этой задачи будет отменен.

Эта функция не меняет состояние не указанных задач.

Пример:
begin
  WizardSelectTasks('!desktopicon');
  // the 'desktopicon' task is deselected
end;
