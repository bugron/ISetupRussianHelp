Pascal Scripting: TaskDialogMsgBox

Прототип:

function TaskDialogMsgBox(const Instruction, TaskDialogText, MsgBoxText: String; const Typ: TMsgBoxType; const Buttons: Cardinal; const ButtonLabels: TArrayOfString; const ShieldButton: Integer; const ForceMsgBox: Boolean): Integer;

Описание:

Показывает диалоговое окно задач, если система это поддерживает и если ForceMsgBox=False. Иначе, показывает обычное окно сообщения.

При диалоговом окне задач:
Instruction указывает инструкцию.
TaskDialogText указывает сообщение.
Typ указывает какую иконку показать в диалоговом окне задач. При значении mbConfirmation иконка не будет отображена.
Buttons указывает какие кнопки включить в диалоговом окне задач.
ButtonLabels указывает, какие метки пользовательских кнопок использовать. При пустом массиве будут использоваться системные метки кнопок по умолчанию. Если метка состоит из двух строк, разделенных новой строкой, то первая строка указывает метку кнопки, а вторая строка - примечание кнопки.
ShieldButton определяет на какой кнопке отображать значок щита. При значении 0 значок щита отображаться не будет.

При обычном окне сообщений:
Instruction указывает заголовок окна. Если задана пустая строка, будет отображаться заголовок по умолчанию.
MsgBoxText указывает сообщение окна.
Typ указывает какую иконку показать в окне сообщений.
Buttons указывает какие кнопки включить в окне сообщений.

Возвращает ID* константу, указывающую кнопку, по которой щелкнул пользователь, или 0, если функция не отработала успешно (что не должно происходить, если не указан неверный параметр или не исчерпаны системные ресурсы.

Примечания:

TMsgBoxType определен как:

TMsgBoxType = (mbInformation, mbConfirmation, mbError, mbCriticalError);

Поддерживаемые значения Buttons:

MB_OK, MB_OKCANCEL, MB_YESNOCANCEL, MB_YESNO, MB_RETRYCANCEL, MB_ABORTRETRYIGNORE

При значении MB_ABORTRETRYIGNORE параметр ButtonLabels не может быть пустым массивом и метки кнопок должны быть указаны в следующем специальном порядке: Повторить, Игнорировать, Прервать.

Поддерживаемые значения ShieldButton и возможные возвращаемые значения:

IDOK, IDCANCEL, IDRETRY, IDYES, IDNO, IDABORT, IDIGNORE

Пример:
begin
  case TaskDialogMsgBox('Choose A or B',
                        'You can choose A or B.', 'You can choose A or B'#13#10#13#10'Do you choose A?',   
                        mbInformation,
                        MB_YESNOCANCEL, ['I choose A'#13#10'A will be chosen.', 'I choose B'#13#10'B will be chosen.'],
                        IDYES, False) of
    IDYES: MsgBox('You chose A.', mbInformation, MB_OK);
    IDNO: MsgBox('You chose B.', mbInformation, MB_OK);
  end;
end;
