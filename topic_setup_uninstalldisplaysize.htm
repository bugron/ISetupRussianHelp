[Setup]: UninstallDisplaySize

Описание:

На Windows 7 инсталлятор использует эту директиву, чтобы установить значение ключа реестра удаления программы EstimatedSize (конечно, если это возможно, а это возможно начиная с Windows 7) Установка и удаление программ Панели управления (на Windows 7 - Программы и компоненты), чтобы автоматически не вычислять его размер. Если директива UninstallDisplaySize не определена, инсталлятор считывает размер непосредственно, беря размер всех установленных файлов и добавление любого установленного значения ExtraDiskSpaceRequired. Примечание: Windows 7 без пакетов обновлений (Service Pack) может показать размер только меньше 4ГБ.

Значение должно быть установлено в байтах. (1048576 байтов = 1 МБ)

Например:
UninstallDisplaySize=1073741824
