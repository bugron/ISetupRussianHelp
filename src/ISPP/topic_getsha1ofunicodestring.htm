Inno Setup Preprocessor: GetSHA1OfUnicodeString

Прототип функции

str GetSHA1OfString(str)


Описание

Возвращает строковое значение контрольной суммы SHA-1 в кодировке Unicode для указанной строки.

Внимание! Компиляцию сценария, в котором используется вызов этой функции, необходимо выполнять в Unicode версии Inno Setup, иначе будет получена внутренняя ошибка.
