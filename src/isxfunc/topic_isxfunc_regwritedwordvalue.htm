Pascal Scripting: RegWriteDWordValue

Прототип:

function RegWriteDWordValue(const RootKey: Integer; const SubKeyName, ValueName: String; const Data: Cardinal): Boolean;

Описание:

Записывает указанное значение реестра типа REG_DWORD. При успешном выполнении возвращает True, иначе False.

Например:
begin
  RegWriteDWordValue(HKEY_CURRENT_USER, 'Software\My Company\My Program',
    'ShowToolbar', 1);
end;
