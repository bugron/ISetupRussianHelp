Pascal Scripting: GetSaveFileName

Прототип:

function GetSaveFileName(const Prompt: String; var FileName: String; const InitialDirectory, Filter, DefaultExtension: String): Boolean;

Описание:

Отображает диалоговое окно, которое позволяет пользователю выбрать новый файл. Возвращает True, пользователь выбрал файл, иначе False. Имя выбранного файла возвращается в строке FileName.

Комментарии:

Пример фильтра: 'Текстовые файлы(*.txt)|*.txt|Все файлы (*.*)|*.*'

Например:
var
  Filename: String;
begin
  // Set the initial filename
  Filename := '';
  if GetSaveFileName('', Filename, '',
     'Text Documents (*.txt)|*.txt|All Files|*.*', 'txt') then
  begin
    // Successful; user clicked OK
    // Filename contains the selected filename
  end;
end;
