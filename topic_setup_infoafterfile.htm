[Setup]: InfoAfterFile

Описание:

Определяет файл в формате .txt или .rtf, который отображается после успешной установки. Этот файл должен быть расположен в исходной папке скрипта, если только для него не указан полный путь или нет префикса "compiler:" - в этом случае компилятор будет искать указанный файл в своем каталоге.

Отличается от файлов с флагом isreadme тем, что отображается в окне инсталлятора, а не в Блокноте.

Если пользователь выберет язык, для которого существует параметр InfoAfterFile, эта директива будет игнорирована. Для справки обратитесь к описанию секции [Languages].

Например:
InfoAfterFile=infoafter.txt
