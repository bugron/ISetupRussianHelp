Pascal Scripting Extended: Pi

Прототип:

function Pi: Extended;

Описание:
Функция Pi возвращает значение с плавающей запятой, дающее полезное приближение значения Пи. Длина окружности = Пи * Диаметр.
Пример:

var
 myPi : Extended;
 diam : Integer;

begin
 diam := 10;

// Показ длины окружности и площади круга
// с диаметром - 10 см
 ShowMessage('Диаметр = '+IntToStr(diam));
 ShowMessage('Длина окружности = '+FloatToStr(Pi*diam));
 ShowMessage('Площадь = '+FloatToStr(Pi*(diam/2)*(diam/2)));
end;


 Диаметр = 10
 Длина окружности = 31.4159265358979
 Площадь = 78.5398163397448
