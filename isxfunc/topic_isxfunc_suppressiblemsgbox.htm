Pascal Scripting: SuppressibleMsgBox

Прототип:

function SuppressibleMsgBox(const Text: String; const Typ: TMsgBoxType; const Buttons, Default: Integer): Integer;

Описание:

Показывает подавляемое окно сообщения. Если окна сообщений подавлены (см. параметры командной строки инсталлятора), возвращается значение переменной Default. Иначе, SuppressibleMsgBox действует как обычный MsgBox.
