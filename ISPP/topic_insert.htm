Inno Setup Preprocessor: #insert

Синтатсис

insert-directive: insert <expr>

Описание

Изменяет точку вставки. По умолчанию, каждая обработанная строка добавляется в конец перевода. Используя insert, точка, в которой следующая обработанная строка будет добавлена к переводу, может быть изменена. insert берет выражение, которое должно оценить к целому числу. Точка вставки будет установлена в это целое число.

Точка вставки также всегда автоматически постепенно увеличивается каждый раз после того, как строка была добавлена к переводу, так, чтобы каждая новая строка была вставлена после ранее вставленного.

Не рекомендуется использовать скрипт, содержащий функцию (такой как SetSetupSetting) который может вставить строку, таким образом, смещая часть преобразования на одну строку вниз, при этом точка вставки не обновляется. Это может привести отличной от ожидаемой результату.

Функция Find функция может использоваться для произведения значений для директивы insert.

Примеры

#insert FindSectionEnd("Icons")
#insert FindSection("Setup") + 1
#insert Find(0, "somefile.ext", FIND_CONTAINS)
