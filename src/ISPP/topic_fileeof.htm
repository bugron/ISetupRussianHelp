Inno Setup Preprocessor: FileEof

Прототип функции

int FileEof(int)


Описание

Возвращает нулевое значение, если указатель файла не установлен в позиции конца файла. Если функция вернула ненулевое значение, то последующие вызовы функции FileRead будут завершены ошибкой. Принимается только дескриптор файла, который возвращает функция FileOpen.
