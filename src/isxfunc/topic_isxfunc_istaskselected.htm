Pascal Scripting: IsTaskSelected

Прототип:

function IsTaskSelected(const Tasks: String): Boolean;

Описание:

Возвращает True, если заданное задание выделено. Несколько заданий могут быть указаны в том же порядке с помощью параметра секции Tasks.

Например:
begin
  if IsTaskSelected('desktopicon') then
    // the 'desktopicon' task is selected
end;
