Pascal Scripting Extended: DayOfWeek

Прототип:

function DayOfWeek(Date: TDateTime): Integer;

Описание:
Функция DayOfTheWeek возвращает индекс дня недели:

 1 = Понедельник
 2 = Вторник
 3 = Среда
 4 = Четверг
 5 = Пятница
 6 = Суббота
 7 = Воскресенье

Пример:

var
 myDate : TDateTime;
 day : string;

begin
 myDate := EncodeDate(2002, 12, 31);

 day := LongDayNames[DayOfWeek(myDate)];

 ShowMessage('Рождество 2002г. '+day);
end;


 Рождество 2002г. Вторник
