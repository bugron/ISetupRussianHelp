Pascal Scripting: WizardDirValue

Прототип:

function WizardDirValue: String;

Описание:

Возвращает текущее содержимое контроля Edit на странице мастера Выбор папки назначения.

В отличие от ExpandConstant('{app}'), эта функция не будет выполнена, если она вызвана после того как мастер появился, но должна быть вызвана до выбора папки пользователем. Скорее всего, она будет возвращать имя каталога по умолчанию.
