Pascal Scripting: BrowseForFolder

Прототип:

function BrowseForFolder(const Prompt: String; var Directory: String; const NewFolderButton: Boolean): Boolean;

Описание:

Отображает диалоговое окно, которое позволяет пользователю выбрать папку. Текущее значение Directory используется как изначально выбранная папка. Если NewFolderButton является True, пользователь с помощь кнопки Новая папка сможет создать новую папку. Возвращает True, если пользователь выбрал папку и нажал OK, иначе False. Выбранная папка возвращается в строке Directory.

Комментарии:

В версиях Windows до ??XP, даже при значении False параметра NewFolderButton кнопка Новая папка всегда будет показана. Это ограничение Windows.
