Pascal Scripting: PageIndexFromID

Прототип:

function PageIndexFromID(const ID: Integer): Integer;

Описание:

Получая ID страницы, функция возвращает индекс ее позиции. Используйте данную функцию, если, например, хотите проверить находится ли данная страница перед какой-нибудь другой.

Пример:
function ShouldSkipPage(PageID: Integer): Boolean;
begin
  Result := PageIndexFromID(PageID) < PageIndexFromID(wpReady);
end;
