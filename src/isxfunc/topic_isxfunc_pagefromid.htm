Pascal Scripting: PageFromID

Прототип:

function PageFromID(const ID: Integer): TWizardPage;

Описание:

Получая ID страницы, функция возвращает ее экземпляр TWizardPage. Используйте данную функцию, если, например, нужно получить доступ к параметрам поверхности страницы зная только ее ID.

При передаче недопустимого ID генерируется исключение.

Например:
var
  Page: TWizardPage;
begin
  Page := PageFromID(wpWelcome);
  Page.Surface.Color := clBlue;
end;
