[Setup]: ReserveBytes

Значение по умолчанию:
0
Описание:

Определяет минимальный размер пустого пространства на первом диске в байтах для установки с разбиением на диски. Это может быть полезно, если понадобится на первый диск скопировать файлы, не являющиеся собственно файлами инсталлятора (например, файл ReadMe).

Компилятор округляет это число до кратности кластера.

Директива игнорируется, если DiskSpanning секции [Setup] равно no.
