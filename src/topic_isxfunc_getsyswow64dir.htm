Pascal Scripting: GetSysWow64Dir

Прототип:

function GetSysWow64Dir: String;

Описание:

Только для 64-битных Windows: возвращает полный путь папки SysWOW64. Это папка, в котором находятся 32-битные системные файлы. Возвращается пустая строка, если эта функция вызывается на 32-битной Windows, или если по какой-то причине не удается получить путь на 64-разрядной ОС Windows (что маловероятно).

Пример:
var
  S: String;
begin
  S := GetSysWow64Dir;
  // Must check the result -- an empty string is returned
  // if there is no SysWOW64 directory.
  if S <> '' then
    MsgBox('SysWOW64 directory: ' + S, mbInformation, MB_OK)
  else
    MsgBox('There is no SysWOW64 directory.', mbInformation, MB_OK);
end;
