Inno Setup Preprocessor: FileRead

Прототип

str FileRead(int)

Описание

Функция читает следующую строку в текстовом файле, открытом функцией FileOpen. Единственный параметр должен быть дескриптором файла, возвращенным функцией FileOpen.

Example

#define FileHandle
#define FileLine
#sub ProcessFileLine
  #define FileLine = FileRead(FileHandle)
  #pragma message FileLine
#endsub
#for {FileHandle = FileOpen("c:\autoexec.bat"); \
  FileHandle && !FileEof(FileHandle); ""} \
  ProcessFileLine
#if FileHandle
  #expr FileClose(FileHandle)
#endif

См. также:

define, sub, pragma, for, if.
