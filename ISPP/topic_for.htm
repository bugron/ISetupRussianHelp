Inno Setup Preprocessor: #for

Синтатсис

for-directive: for   { <expr1> ; <expr2> ; <expr3> } <expr4>

Описание

Используйте директиву for, чтобы получить цикл. for получает 4 выражения. Первое выражение (expr1) называется "выражение инициализации," второе выражение (expr2) "условие," третье выражение (expr3) "действие," и последнее выражение (expr4) "тело цикла."

Логика директивы for:

  1. Оценивается выражение инициализации.

  2. Оценивается условие. Если это 0, цикла останавливается.

  3. Оценивается тело цикла.

  4. Оценивается действие.

  5. Процесс повторяется.

Примеры

// Call AddFile user defined procedure 200 times
#for {i = 200; i > 0; i--} AddFile
