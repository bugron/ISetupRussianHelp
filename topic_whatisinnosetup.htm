Inno Setup - бесплатный инсталлятор для программ Windows. Первая версия этой программы появилась в 1997 и сегодня Inno Setup превосходит многие коммерческие инсталляторы по стабильности и возможностям.

Основные достоинства:

  • Поддержка всех версий Windows начиная с 2000, включая: Windows 10, Windows 8.1, Windows 8, Windows Server 2012, Windows 7, Windows Server 2008 R2, Windows Vista, Windows Server 2008, Windows XP, Windows Server 2003, и Windows 2000 (Пакеты обновлений не требуются.)
  • Обширная поддержка установки 64-разрядных приложений на 64-разрядных версиях Windows. Поддерживаются архитектуры x64 и Itanium. (Для работы 64-разрядного режима на архитектуре Itanium требуется пакет обновления 1 или поздние версии Windows Server 2003.)
  • Возможность создания одного EXE файла-инсталлятора, что упрощает online-установку программы. Также поддерживается разбивка на диски.
  • Стандартный стиль интерфейса мастера Windows.
  • Задание типов установки: Полная, Минимальная, Выборочная.
  • Создание деинсталлятора.
  • Установка файлов.
  • Встроенное сжатие файлов (bzip2 и 7-Zip, LZMA/LZMA2). Инсталлятор может сравнивать версии файлов, заменять загруженные в память файлы, вести подсчет общедоступных файлов, установить шрифты и т.д.
  • Создание ярлыков в любом месте, включая меню Пуск и Рабочий стол.
  • Работа с реестром и INI файлами.
  • Запуск других программ перед установкой, в процессе и после установки.
  • Поддержка многоязычных инсталляторов, включая поддержку языков с письмом с права налево.
  • Возможность защиты инсталлятора паролем и шифрованием.
  • Создание инсталляторов и деинсталляторов с цифровой подписью.
  • Тихая установка и тихое удаление.
  • Создание инсталляторов с поддержкой Unicode
  • Встроенный компилятор языка Pascal для усовершенствованной настройки параметров установки и деинсталляции.
  • Возможен просмотр исходного кода создаваемого инсталлятора (Borland Delphi 2.0-5.0 и 2009).


Inno Setup действительно полностью бесплатная программа, даже для коммерческого использования?

Да, Inno Setup может использоваться абсолютно бесплатно, даже для коммерческого применения.

(Примечание: не следует путать "бесплатность" и "общедоступность" программы. Inno Setup защищен авторским правом, и использование исходного кода самого Inno Setup нежелательно. Подробнее о правилах установки и использования можно прочитать в файле лицензии LICENSE.TXT.)
