Inno Setup Preprocessor: Int

Прототип

int Int(any 1, int? 2)

Описание

Функция преобразовывает выражение (первый параметр) к его целочисленному представлению. Если выражение - целое число, результат функции - значение выражения. Если выражение является пустым, результат 0. Если выражение - строка, ISPP пытается преобразовать ее в целое число; если такая попытка проваливается, выдается ошибка, (если второй параметр не определяет результат по умолчанию).
