Pascal Scripting: RenameFile

Прототип:

function RenameFile(const OldName, NewName: string): Boolean;

Описание:

Переименовывает файл или папку с OldName в NewName. При успешном выполнении, возвращает True. Если не удается изменить имя файла (например, файл с именем NewName уже существует), то функция возвращает значение False.
