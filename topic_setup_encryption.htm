[Setup]: Encryption

Возможные значения:
yes или no

Значение по умолчанию:
no
Описание:

При значении yes, файлы инсталлятора (параметры секции [Files]) будут зашифрованы с помощью ARCFOUR с 128-битным ключом, сгенерированным из директивы Password секции [Setup].

Поддержка шифрования не включена в Inno Setup напрямую, библиотеку нужно скачать и установить отдельно. Информацию об этом можно найти на английском сайте программы в разделе Inno Setup Downloads.

Если шифрование включено, и вы вызовете функцию ExtractTemporaryFile из секции [Code], а пользователь введет правильный пароль, функция не будет работать до тех пор, пока в секции [Files] для файла не будет установлен флаг noencryption.

Ключ, используемый при шифровании - 160-битный SHA-1/128-битный MD5 хеш 64-битной случайной последовательности плюс значение пароля.
