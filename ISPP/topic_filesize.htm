Inno Setup Preprocessor: FileSize

Прототип

int FileSize(str)

Описание

Возвращает размер указанного файла в байтах. Если файл не существует, результат равно 1. ISPP поддерживает только целые 32 битные числа со знаком: для файлов, больше чем 2 Гбайт (и меньше чем 4 Гбайт), результат отрицателен, так что имейте это ввиду.
