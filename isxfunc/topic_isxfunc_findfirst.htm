Pascal Scripting: FindFirst

Прототип:

function FindFirst(const FileName: String; var FindRec: TFindRec): Boolean;

Описание:

Получает информацию о первом файле, который соответствует специальному символу, указанному в FileName. При успешном выполнении возвращает True.

TFindRec определен так:

  TFindRec = record
    Name: String;               // имя найденного фале (не путь)
    Attributes: LongWord;       // атрибуты файла
    SizeHigh: LongWord;         // размер файла, upper 32 bits
    SizeLow: LongWord;          // размер файла, lower 32 bits
    CreationTime: TFileTime;    // время создания файла
    LastAccessTime: TFileTime;  // время последнего доступа к файлу
    LastWriteTime: TFileTime;   // время последнего изменения файла
    AlternateName: String;      // короткое имя файла
    FindHandle: THandle;        // для внутреннего использования
  end;

TFileTime определен так:

  TFileTime = record
    dwLowDateTime: DWORD;
    dwHighDateTime: DWORD;
  end;

Атрибуты файлов:

  FILE_ATTRIBUTE_READONLY
  FILE_ATTRIBUTE_HIDDEN
  FILE_ATTRIBUTE_SYSTEM
  FILE_ATTRIBUTE_DIRECTORY
  FILE_ATTRIBUTE_ARCHIVE
  FILE_ATTRIBUTE_DEVICE
  FILE_ATTRIBUTE_NORMAL
  FILE_ATTRIBUTE_TEMPORARY
  FILE_ATTRIBUTE_SPARSE_FILE
  FILE_ATTRIBUTE_REPARSE_POINT
  FILE_ATTRIBUTE_COMPRESSED
  FILE_ATTRIBUTE_OFFLINE
  FILE_ATTRIBUTE_NOT_CONTENT_INDEXED
  FILE_ATTRIBUTE_ENCRYPTED
Комментарии:

Если FindFirst возвращает True, вызовите FindNext для перечисления остальных файлов, а затем вызовите FindClose.

Например:

В примере считывается число файлов в папке {sys}.

var
  FilesFound: Integer;
  FindRec: TFindRec;
begin
  FilesFound := 0;
  if FindFirst(ExpandConstant('{sys}\*'), FindRec) then begin
    try
      repeat
        // Don't count directories
        if FindRec.Attributes and FILE_ATTRIBUTE_DIRECTORY = 0 then
          FilesFound := FilesFound + 1;
      until not FindNext(FindRec);
    finally
      FindClose(FindRec);
    end;
  end;
  MsgBox(IntToStr(FilesFound) + ' files found in the System directory.',
    mbInformation, MB_OK);
end;
