Pascal Scripting: LoadDLL

Прототип:

function LoadDLL(const DLLName: String; var ErrorCode: Integer): Longint;

Описание:

Загружает указанный DLL. Возвращает дескриптор DLL, если DLL был загружен успешно, иначе возвращает 0. ErrorCode указывает на возникшую ошибку, если возвращено значение 0. Используйте SysErrorMessage(ErrorCode), чтобы получить описание ошибки.

Комментарии:

Эта функция устарела. См. использование DLL.

Ссылка на коды системных ошибок в MSDN.
