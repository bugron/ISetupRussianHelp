Inno Setup Preprocessor: Функции

Есть много предопределенных функций, обеспеченных ISPP, которые можно использовать, чтобы выполнить действия во время компиляции и/или изменить Ваш скрипт. Например, прочитать информацию о версии EXE файла и использовать возвращаемое значение функции, чтобы изменить скрипт:

#define MyAppVer GetFileVersion(AddBackslash(SourcePath) + "MyProg.exe")

[Setup]
AppVerName=MyProg version {#MyAppVer}
