Inno Setup Preprocessor: Exec

Прототип функции

int Exec(str 1, str? 2, str? 3, int? 4, int? 5)


Описание

Выполняет запуск указанного исполняемого файла.

Первый аргумент определяет имя файла, который необходимо запустить.

Второй аргумент может быть использован для указания параметров командной строки, которые необходимо передать исполняемому файлу.

Третий аргумент может быть использован для указания рабочего каталога, в котором выполняется запуск указанного файла.

Четвёртый аргумент используется для указания статуса запущенного процесса. Если не требуется ожидать завершение работы запущенного процесса, то значение аргумента должно быть установлено в ноль (0). По умолчанию принято ненулевое значение.

Пятый аргумент является флагом (SW_*), который характеризует режим вывода графической оболочки (GUI) запускаемого процесса. Каждому флагу соответствует определенные константы, которые объявлены в файле ISPPBuiltins.iss. По умолчанию, для запуска всех GUI процессов, принимается флаг SW_SHOWNORMAL (т.е. 1) - инициализация окна и вывод его на экран.

Если четвертый аргумент не указан, или имеет ненулевое значение, то функция возвращает код завершения запущенного процесса. Если для аргумента было задано нулевое значение, то результат функции указывает на характер запуска указанного процесса: неудача (функция вернёт ноль) или успех (функция вернёт отличное от нуля значение).
