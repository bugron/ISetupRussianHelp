Pascal Scripting: GetShellFolder

Прототип:

function GetShellFolder(Common: Boolean; const ID: TShellFolderID): String;

Описание:

Получает путь к указанной папке оболочки. Если Common равен True и пользователь обладает правами администратора, то возвращает путь к так называемой 'общей' версии папки оболочки. При неудаче (маловероятно, но возможно), возвращается пустая строка.

Комментарии:

Существует мало оснований для использования этой функции. Рекомендуется использовать функцию ExpandConstant, чтобы получить пути к папкам оболочки.

Например:
var
  Path: String;
begin
  Path := GetShellFolder(False, sfAppData);
  if Path <> '' then
  begin
    MsgBox('Application Data path = ' + Path, mbInformation, MB_OK);
  end
  else
  begin
    // handle failure
  end;
end;
