Pascal Scripting: Delete

Прототип:

procedure Delete(var S: String; Indx, Count: Integer);

Описание:

Удаляет подстроку длиной Count в строке S, начинающийся с позиции S[Index].
Если Index больше чем длина строки S, символы не удаляются. Если Count определяет больше символов, чем есть с места S[Index], Delete удаляет остальную часть строки.
