Pascal Scripting: VarType

Прототип:

function VarType(const V: Variant): TVarType;

Описание:

Returns the type code of the specified variant.

Младшие двенадцать битов кода типа (они определены маской varTypeMask) определяют тип переменной. Если переменная является массивом указанного типа, то стоит бит varArray. Если переменная является ссылкой значения указанного типа вместо реального значения, то стоит бит varByRef.

Определены следующие коды типов:

const
  varEmpty    = $0000;
  varNull     = $0001;
  varSmallint = $0002;
  varInteger  = $0003;
  varSingle   = $0004;
  varDouble   = $0005;
  varCurrency = $0006;
  varDate     = $0007;
  varOleStr   = $0008;
  varDispatch = $0009;
  varError    = $000A;
  varBoolean  = $000B;
  varVariant  = $000C;
  varUnknown  = $000D;
  varByte     = $0011;
  varString   = $0100;
  varTypeMask = $0FFF;
  varArray    = $2000;
  varByRef    = $4000;
