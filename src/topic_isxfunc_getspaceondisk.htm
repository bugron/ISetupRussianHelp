Pascal Scripting: GetSpaceOnDisk

Прототип:

function GetSpaceOnDisk(const Path: String; const InMegabytes: Boolean; var Free, Total: Cardinal): Boolean;

Описание:

Возвращает количество свободного и общего байтов или мегабайт на диске. Path указывает папку на диске или UNC путь; это может быть как корень диска (например C:\), так и вложенная папка. Параметр InMegabytes определяет данные возвращаются в байтах, или мегабайтах (2^20), округленных в меньшую сторону. При удачном выполнении возвращает True, иначе False.

Комментарии:

Возвращаемое число ограничено: 2147483647 (2^31-1). Поэтому, если InMegaBytes является False, функция возвращает значение не больше 2147483647 байтов. Если InMegaBytes является True, возвращает значение не больше 2147483647 мегабайтов.

Пример:
var
  Path: String;
  FreeMB, TotalMB: Cardinal;
begin
  // Get and display free megabytes on the Program Files drive
  Path := ExpandConstant('{autopf}');
  if GetSpaceOnDisk(Path, True, FreeMB, TotalMB) then
  begin
    MsgBox('There are ' + IntToStr(FreeMB) + ' megabytes free on ' +
      Path, mbInformation, MB_OK);
  end
  else begin
    // the function failed
  end;
end;
