Pascal Scripting: GetOpenFileName

Прототип:

function GetOpenFileName(const Prompt: String; var FileName: String; const InitialDirectory, Filter, DefaultExtension: String): Boolean;

Описание:

Отображает диалоговое окно, которое позволяет пользователю выбрать существующий файл. Возвращает True, пользователь выбрал файл, иначе False. Имя выбранного файла возвращается в строке FileName.

Комментарии:

Пример фильтра: 'Текстовые файлы(*.txt)|*.txt|Все файлы (*.*)|*.*'

Например:
var
  FileName: String;
begin
  // Set the initial filename
  FileName := '';
  if GetOpenFileName('', FileName, '',
     'Text Documents (*.txt)|*.txt|All Files|*.*', 'txt') then
  begin
    // Successful; user clicked OK
    // FileName contains the selected filename
  end;
end;
