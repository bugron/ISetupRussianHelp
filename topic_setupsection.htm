Секция [Setup]

Параметры этой секции глобальны, т. е. задают общие параметры для инсталлятора и деинсталлятора. Некоторые директивы данной секции необходимы для создания любого инсталлятора. Пример секции [Setup]:

[Setup]
AppName=My Program
AppVersion=1.5
DefaultDirName={pf}\My Program
DefaultGroupName=My Program

По умолчанию, все начальные и конечные пробелы в значении директивы будут удалены. Можно избежать этого, взяв значение директивы в двойные кавычки (").
