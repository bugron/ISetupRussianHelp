Pascal Scripting: CallDLLProc

Прототип:

function CallDLLProc(const DLLHandle: Longint; const ProcName: String; const Param1, Param2: Longint; var Result: Longint): Boolean;

Описание:

Вызывает указанную функцию в DLL файле, используя дескриптор этого DLL, полученный функцией LoadDLL. При успешном вызове возвращает True, иначе False.
Функция должна использовать стандартное соглашение о вызовах (stdcall), принимать два 4-х байтных целочисленных параметра и возвращать 4-х байтный целочисленный результат.

Комментарии:

Эта функция устарела. См. использование DLL.
