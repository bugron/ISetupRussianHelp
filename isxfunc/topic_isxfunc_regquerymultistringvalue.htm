Pascal Scripting: RegQueryMultiStringValue

Прототип:

function RegQueryMultiStringValue(const RootKey: Integer; const SubKeyName, ValueName: String; var ResultStr: String): Boolean;

Описание:

Запрашивает значение реестра типа REG_MULTI_SZ и возвращает данные в переменной ResultStr. При успешном выполнении возвращает True. При значении False ResultStr не меняется.

Комментарии:

In a REG_MULTI_SZ-type value, each string is separated by a null character (#0).
