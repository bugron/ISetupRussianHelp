[Setup]: DefaultGroupName

Описание:

Директива определяет имя группы меню Пуск по умолчанию, которое используется на странице мастера Выбор программной группы. Если значение директивы не указано или пустое, будет использовано значение "По умолчанию" "(Default)".

Помните, что для групп меню Пуск действуют правила имен папок, отличные от правил для обычных папок.

Например:
DefaultGroupName=My Program
