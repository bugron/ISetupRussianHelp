Pascal Scripting: CreateOutputMsgPage

Прототип:

function CreateOutputMsgPage(const AfterID: Integer; const ACaption, ADescription, AMsg: String): TOutputMsgWizardPage;

Описание:

Создает страницу мастера, содержащую только статический текст. Параметр AMsg определяет текст, который нужно вывести на экран.

Например:
var
  Page: TOutputMsgWizardPage;

...

// Create the page
Page := CreateOutputMsgPage(wpWelcome,
  'Information', 'Please read the following important information before continuing.',
  'Blah blah blah.');
