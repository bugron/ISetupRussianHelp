Pascal Scripting: GetIniInt

Прототип:

function GetIniInt(const Section, Key: String; const Default, Min, Max: Longint; const Filename: String): Longint;

Описание:

Читает двойное целое (Longint) из INI файла. Если полученное значение не находится между Min/Max, тогда функция возвращает Default. Если Min=Max тогда значения Min и Max игнорируются.
