[Setup]: LZMAMatchFinder

Возможные значения:
HC или BT

Значение по умолчанию:
HC, если уровень сжатия LZMA установлен в fast,
иначе BT
Описание:

Управляет методом средства обнаружения соответствия, используемым компрессорами LZMA и LZMA2.

Значение HC включает метод Hash Chain с 4 байтами хеша.

Значение BT включает метод Binary Tree с 4 байтами хеша.

Метод Binary Tree может обеспечить лучшую степень сжатия за счет скорости сжатия.
