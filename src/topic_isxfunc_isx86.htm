Pascal Scripting: IsX86

Прототип:

function IsX86: Boolean;

Описание:

Возвращает True, если процессор имеет архитектуру x86.

Эта функция является сокращением для ProcessorArchitecture. Перед использованием этой функции, прочтите ProcessorArchitecture.
