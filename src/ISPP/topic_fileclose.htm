Inno Setup Preprocessor: FileClose

Прототип функции

void FileClose(int)


Описание

Выполняет закрытие открытого файла и всех ресурсов, которые были выделены при вызове функции FileOpen. После вызова FileClose, дескриптор файла становится недействительным.

Эта функция является устаревшей, т.к. начиная с версии 1.2. ISPP автоматически освобождает ресурсы, выделенные при вызове FileOpen.
