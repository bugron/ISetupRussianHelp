[Setup]: WizardImageFile

Значение по умолчанию:
compiler:WIZMODERNIMAGE.BMP
Описание:

Определяет картинку в формате .bmp, которая будет отображаться в левой части окна инсталлятора. Этот файл должен быть расположен в исходной папке скрипта, если только для него не указан полный путь или нет префикса "compiler:" - в этом случае компилятор будет искать указанный файл в своем каталоге.

256-цветные картинки могут не отобразиться правильно в 256-цветном режиме из-за скудности палитры. Максимальный размер картинки 164x314 пикселей. Обратите внимание, что если в Windows задан крупный шрифт, площадь для картинки в окне мастера будет больше.

Например:
WizardImageFile=myimage.bmp
