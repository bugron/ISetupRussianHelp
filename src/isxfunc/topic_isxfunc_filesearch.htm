Pascal Scripting: FileSearch

Прототип:

function FileSearch(const Name, DirList: string): String;

Описание:

Ищет в каталогах DirList файл под названием Name. DirList - список каталогов, разделенных точками с запятой. Если FileSearch определяет местоположение файла Name, он возвращает строку, которая содержит полностью определенный путь файла Name. Если нет совпадений, FileSearch возвращает пустую строку.
