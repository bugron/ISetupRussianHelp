Pascal Scripting: UnloadDLL

Прототип:

procedure UnloadDLL(Filename: String);

Описание:

Выгружает заданный DLL, который был загружен из секции [Code], используя ключевое слово "external". Это может быть полезно, если вы хотите удалить DLL.
Имя файла и путь должны точно совпадать. Вы должны развернуть любые константы в имени файла самостоятельно прежде, чем передать их UnloadDLL.
If the function import used a "files:" prefix, prepend the value of the {tmp} constant to the filename (e.g. ExpandConstant('{tmp}\filename.dll')).

Комментарии:

Не рекомендуется попробовать, но если Вы попытаетесь вызвать функцию в DLL, который уже выгружен, то DLL будет перезагружен.

Например:
procedure DllFunc; external 'DllFunc@{app}\MyDll.dll stdcall uninstallonly';

...

begin
  // Call DllFunc
  DllFunc;

  // Unload the DLL
  UnloadDLL(ExpandConstant('{app}\MyDll.dll'));

  // Now we can delete the DLL
  DeleteFile(ExpandConstant('{app}\MyDll.dll'));
end;
