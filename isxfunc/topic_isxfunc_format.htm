Pascal Scripting: Format

Прототип:

function Format(const Format: string; const Args: array of const): string;

Описание:

Форматирует ряд аргументов в открытом массиве Args. Форматирование управляется строкой Format; результат возвращается в виде строки.

Исключение будет поднято, если указан недопустимый формат строки, переданы слишком мало аргументов, или если любой из аргументов имеет неправильный тип.

Например:
var
  S: String;
  I: Integer;
begin
  S := Format('%d files found', [10]);
  // S = '10 files found'

  S := Format('Filename: %s', ['file.txt']);
  // S = 'Filename: file.txt'

  I := 64;
  S := Format('%d in hex, padded to 8 digits: %.8x', [I, I]);
  // S = '64 in hex, padded to 8 digits: 00000040'
end;
