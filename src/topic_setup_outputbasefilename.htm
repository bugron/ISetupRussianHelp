[Setup]: OutputBaseFilename

Значение по умолчанию:
mysetup
Описание:

Эта директива позволяет задать конечное имя получаемого инсталлятора, чтобы после создания не приходилось его переименовывать.

Не рекомендуется установить значение setup. Все исполняемые файлы с именем "setup.exe" проходят через сервис совместимости приложений. Таким образом загружаются дополнительные DLL файлы, такие как version.dll. Windows подключает эти DLL файлы небезопасным образом, поэтому в процессе эти файлы могут быть заменены.

Например:
OutputBaseFilename=MyProg100
