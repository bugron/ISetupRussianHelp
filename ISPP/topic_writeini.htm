Inno Setup Preprocessor: WriteIni

Прототип

void WriteIni(str 1, str 2, str 3, any 4)

Описание

Записывает определенное значение в INI файл. Параметром 1 должно быть имя INI файла, параметр 2 - имя раздела в INI файле, третий параметр - ключ в разделе. Последний параметр должен быть установлен в значение, которое Вы хотите записать в INI файл, это может быть значение любого типа.
