Inno Setup Preprocessor: Delete

Delete

#define Delete(str *S, int Index, int Count = MaxInt)


Описание

Выполняет удаление заданного количества символов MaxInt в строке S, начиная с позиции Index. Строка S передается в виде ссылки, поэтому возможно её изменение.

Функция объявлена в файле ISPPBuiltins.iss.
