Pascal Scripting Extended: Date

Прототип:

function Date: TDateTime;

Описание:
Функция Date передает текущую дату на данном компьютере для локального временного пояса.
 Поскольку возвращаемая величина является типом TDateTime, компонент времени установлен в нуль (начало дня).
Пример:

begin
 ShowMessage('Вчера = '+DateToStr(Yesterday));
 ShowMessage('Сегодня = '+DateToStr(Date));
 ShowMessage('Завтра = '+DateToStr(Tomorrow));

 ShowMessage('Сегодняшнее время = '+TimeToStr(Date));
end;


 Вчера = 28/10/2002
 Сегодня = 29/10/2002
 Завтра = 30/10/2002
 Сегодняшнее время = 00:00:00
