Pascal Scripting: EnableFsRedirection

Прототип:

function EnableFsRedirection(const Enable: Boolean): Boolean;

Описание:

Управляет состоянием перенаправления файловой системы WOW64 для встроенных функций (с некоторыми исключениями). Укажите True в параметре Enable, чтобы оставить включенным перенаправление для вызываемых функций, укажите False для отключения. Функция возвращает предыдущее состояние перенаправления (True, если перенаправление было включено).

Если значение параметра Enable равен False и пользователь не работает в 64-разрядном версии Windows, будет вызвано исключение. Чтобы избежать исключения, сначала вызовите IsWin64.

Комментарии:

После того, как вы выполнили операцию, требующую смены состояния перенаправления, убедитесь, что вернули состояние до выполнения операции. Всегда используйте конструкцию try..finally, для обеспечения восстановления предыдущего состояния при возникновении исключения. См. ниже для примера.

По умолчанию, перенаправление файловой системы включено в 32-разрядном режиме установки и отключено в 64-разрядном режиме установки.

Данная функция не действует для вызовов внешних DLL функций. При вызове внешних функций перенаправление файловой системы всегда остается включенным.

Можно вызвать функцию со значением параметра Enable равным False, если пользователь не работает в 64-разрядном версии Windows. Но в этом случае этот вызов не будет иметь никакого эффекта.

Пример:

Следующий пример демонстрирует как можно запустить исполняемый файл из 64-разрядной системной папки. (Примечание: В 64-разрядном режиме установки нет необходимости вызова функции EnableFsRedirection, поскольку перенаправление файловой системы уже отключено по умолчанию.)

var
  OldState: Boolean;
  ResultCode: Integer;
begin
  // Сначала убедимся, что пользователь работает в 64-разрядном версии
  // Windows, поскольку иначе вызов EnableFsRedirection(False)
  // приведет в вызову исключения.
  if IsWin64 then
  begin
    // Отключить перенаправление, так что можно запустить cmd.exe
    // из 64-разрядной системной папки.
    OldState := EnableFsRedirection(False);
    try
      Exec(ExpandConstant('{cmd}'), '', '', SW_SHOW,
        ewWaitUntilTerminated, ResultCode);
    finally
      // Восстановить предыдущее состояние перенаправления.
      EnableFsRedirection(OldState);
    end;
  end;
end;
