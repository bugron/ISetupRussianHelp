Pascal Scripting: GetSpaceOnDisk64

Прототип:

function GetSpaceOnDisk64(const Path: String; var Free, Total: Int64): Boolean;

Описание:

Возвращает количество свободного и общего байтов на диске. Path указывает папку на диске или UNC путь; это может быть как корень диска (например C:\), так и подпапка. При удачном выполнении возвращает True, иначе False.

Комментарии:

Требуется Unicode Inno Setup.

Например:
var
  Path: String;
  FreeBytes, TotalBytes: Int64;
begin
  // Get and display free bytes on the Program Files drive
  Path := ExpandConstant('{pf}');
  if GetSpaceOnDisk64(Path, FreeBytes, TotalBytes) then
  begin
    MsgBox('There are ' + IntToStr(FreeBytes) + ' bytes free on ' +
      Path, mbInformation, MB_OK);
  end
  else begin
    // the function failed
  end;
end;
