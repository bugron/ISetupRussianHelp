Inno Setup Preprocessor: ReadReg

Прототип функции

any ReadReg(int 1, str 2, str? 3, any? 4)


Описание

Выполняет чтение значения из указанного ключа реестра (32-бит или 64-бит) операционной системы Windows. Первый аргумент должен быть именем раздела реестра, например, HKEY_LOCAL_MACHINE (32-бит) или HKEY_LOCAL_MACHINE_64 (64-бит). Константы, используемые этим параметром, объявлены в файле ISPPBuiltins.iss. Второй аргумент – это имя подраздела реестра. Третий аргумент является ключом реестра, значение которого необходимо получить. Последний аргумент (4) является необязательным и может быть использован для установки значения, которое вернет функция в случае неудачного выполнения операции.

Заметьте, функция возвращает значение любого типа в зависимости от фактического типа данных в ключе реестра.
