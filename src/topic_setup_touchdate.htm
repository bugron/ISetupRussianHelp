[Setup]: TouchDate

Возможные значения:
current, none, или YYYY-MM-DD

Значение по умолчанию:
current
Описание:

Дата используется в отметке даты/времени для файлов секции [Files] и имеющих флаг touch.

При значении current дата равна системной дате (во время компиляции). При значении none дата оставляется не тронутой. Иначе TouchDate толкуется открыто в формате YYYY-MM-DD (ISO 8601). Если TimeStampsInUTC=yes, дата принимает значение UTC-даты.

Например:
TouchDate=2004-01-31
