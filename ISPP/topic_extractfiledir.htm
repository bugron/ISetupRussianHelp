Inno Setup Preprocessor: ExtractFileDir

ExtractFileDir

#define ExtractFileDir(str PathName)


Описание

Выполняет извлечение пути к папке для заданного файла. При этом путь НЕ будет содержать символа обратной косой черты (кроме корневого каталога). Если PathName не содержит путь или его части, то функция возвращает пустую строку.

Функция объявлена в файле ISPPBuiltins.iss.
