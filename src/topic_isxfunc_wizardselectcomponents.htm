Pascal Scripting: WizardSelectComponents

Прототип:

procedure WizardSelectComponents(const Components: String);

Описание:

Выбирает указанный список компонентов, разделенных запятыми.

Если имя компонента начинается с символа "*", то дочерние компоненты тоже будут выбраны (кроме тех, у которых установлен флаг dontinheritcheck). Если имя компонента начинается с символа "!", то выбор этого компонента будет отменен.

Эта функция не меняет состояние не указанных компонентов. Эта функция может изменить состояние компонент, у которых установлен флаг fixed.

Пример:
begin
  WizardSelectComponents('!helpfiles');
  // the 'helpfiles' component is deselected
end;
