Inno Setup Preprocessor: ExtractFilePath

ExtractFilePath

#define ExtractFilePath(str PathName)


Описание

Выполняет извлечение пути к папке для заданного файла. При этом путь будет содержать символ обратной косой черты. Если PathName не содержит путь к файлу, то функция возвращает пустую строку.

Функция объявлена в файле ISPPBuiltins.iss.
