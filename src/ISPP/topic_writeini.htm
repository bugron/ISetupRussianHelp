Inno Setup Preprocessor: WriteIni

Прототип функции

void WriteIni(str 1, str 2, str 3, any 4)


Описание

Выполняет запись указанного значения в INI-файл. Первый аргумент является именем INI-файла. Второй аргумент – это имя секции в INI-файле. Третий аргумент – имя ключа в указанной секции. Четвертый аргумент должен содержать значение (любого типа), которое необходимо записать в INI-файл.
