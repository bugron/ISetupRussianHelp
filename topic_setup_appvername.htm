[Setup]: AppVerName

Значение по умолчанию:
AppName версия AppVersion, локализованный согласно активному языку пользовательского сообщения NameAndVersion
Описание:

Значение этой обязательной директивы должно быть таким же, как AppName, но с версией программы. Значение может содержать константы.

Эта директива требуется, если директива AppVersion не установлена.

Примеры:
AppVerName=My Program 1.5
AppVerName=My Program version 1.5
AppVerName={cm:NameAndVersion,My Program,1.5}
