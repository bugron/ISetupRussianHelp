Inno Setup Preprocessor: FileEof

Прототип

int FileEof(int)

Описание

Функция возвращает нуль, если указатель файла не указывает на конец файла, иначе возвращает ненулевое значение. Если эта функция возвращает ненулевое значение, последующий FileRead не будет вызван. Единственный параметр должен быть дескриптором файла, возвращенным FileOpen.
