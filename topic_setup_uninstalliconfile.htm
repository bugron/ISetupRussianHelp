[Setup]: UninstallIconFile

Описание:

Не используется с версии 5.0.0. Задает новую иконку для программы-деинсталлятора. Новая иконка будет отображена в правом верхнем углу окна Процесс удаления. Этот файл должен быть расположен в исходной папке скрипта, если только для него не указан полный путь или нет приставки "compiler:" - в этом случае компилятор будет искать указанный файл в своем каталоге. Эта директива не поддерживается в Windows 95/98/Me.
