[Setup]: OutputDir

Значение по умолчанию:
Output
Описание:

Папка, в которой будут создаваться файлы установки относительно папки скрипта. По умолчанию эта папка создается в папке скрипта и именуется Output.

Если путь OutputDir задан не полностью, компилятор сделает папку дочерней относительно папки скрипта.

Например:
OutputDir=c:\output
