Pascal Scripting Extended: Now

Прототип:

function Now: TDateTime;

Описание:
Функция Now возвращает текущую дату и время для местной временной зоны.
Возвращенный тип TDateTime может быть использован и показан многими SysUtil и DateUtils процедурами.

Пример:

var
 today : TDateTime;

begin
 today := Now;
 ShowMessage('День = '+DateToStr(today));
 ShowMessage('Время = '+TimeToStr(today));
end;


 День = 05/01/2007
 Время = 13:37:25
