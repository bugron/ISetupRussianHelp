Pascal Scripting: FileSize

Прототип:

function FileSize(const Name: String; var Size: Integer): Boolean;

Описание:

Устанавливает Size в размер заданного файла в байтах. Возвращает True, операция прошла успешно, иначе False. Поддерживает файлы только меньше 2 ГБ. Чтобы получить все 64 бита размера файла, вместо данной функции используйте FindFirst/FindClose.
