Pascal Scripting: IsWin64

Прототип:

function IsWin64: Boolean;

Описание:

Возвращает True, если система является 64-разрядной версией ОС Windows. При значении False, Вы не можете использовать ни один из 64-разрядных функций Inno Setup.

Не используйте эту функцию для обнаружения 64-разрядного режима инсталляции, для этого используйте Is64BitInstallMode.

Например:
begin
    // Check IsWin64 before using a 64-bit-only feature to
    // avoid an exception when running on 32-bit Windows.
    if IsWin64 then
    begin
      MsgBox('64-bit program files reside in: ' +
        ExpandConstant('{autopf64}'), mbInformation, MB_OK);
    end;
  end;
