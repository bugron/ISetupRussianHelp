Pascal Scripting Extended: Time

Прототип:

function Time: TDateTime;

Описание:
Функция Time возвращает текущее время в местном часовом поясе.
Из-за того, что возвращаемое значение типа TDateTime, дата устанавливается в 0 (30/12/1899).
Пример:

var
 today : TDateTime;

begin
 today := Time;
 ShowMessage('текущая дата = '+DateToStr(today));
 ShowMessage('текущее время = '+TimeToStr(today));
end;


 текущая дата = 30/12/1899
 текущее время = 13:37:25
