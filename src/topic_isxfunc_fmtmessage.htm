Pascal Scripting: FmtMessage

Прототип:

function FmtMessage(const S: String; const Args: array of String): String;

Описание:

Форматирует стоку S используя заданные строковые аргументы. %1 в форматированной строке будет заменен первым значением массива Args; a %2 будет заменен вторым значением; и так далее. %% будет заменен одним символом процента: %.

Комментарии:

Если %-спецификатор ссылается на несуществующий аргумент, он будет возвращен без изменений. Исключение обрабатываться не будет.

Пример:
var
  S: String;
begin
  S := FmtMessage('%1 version %2 will be installed.', ['My Program', '1.0']);
  // S = 'My Program version 1.0 will be installed.'

  S := FmtMessage(SetupMessage(msgNotOnThisPlatform), ['Windows 2000']);
  // S = 'This program will not run on Windows 2000.'
end;
