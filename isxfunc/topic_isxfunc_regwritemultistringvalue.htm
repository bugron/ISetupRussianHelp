Pascal Scripting: RegWriteMultiStringValue

Прототип:

function RegWriteMultiStringValue(const RootKey: Integer; const SubKeyName, ValueName, Data: String): Boolean;

Описание:

Записывает указанное значение реестра типа REG_MULTI_SZ. При успешном выполнении возвращает True, иначе False.

Комментарии:

В значении типа REG_MULTI_SZ каждая строка разделяется нулевым символом (#0).

Например:
begin
  RegWriteMultiStringValue(HKEY_CURRENT_USER, 'Software\My Company\My Program',
    'MultiStringTest', 'String1' + #0 + 'String2' + #0 + 'String3');
end;
