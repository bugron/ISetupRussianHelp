Inno Setup Preprocessor: #define

Синтатсис

определение директив: <определение переменной>
<определение макросов>
<установка области видимости по умолчанию.>
определение переменной: (define | :) [private | protected | public] <ident> [[ <expr> ]] [[=] <expr>]
определение макросов: (define | :) [private | protected | public] <ident> ( [<formal-macro-args>] ) <expr>
установка области видимости по умолчанию: (define | :) private | protected | public
formal-macro-args: <formal-macro-arg> [, <formal-macro-arg>]...
formal-macro-arg: <by-val-arg> | <by-ref-arg>
by-val-arg: [<type-id>] <ident> [= <expr>]
by-ref-arg: [<type-id>] * <ident>
type-id: any | int | str | func

Описание

Первый синтаксис ("определение переменной") определяет переменную с именем ident, или присваивает значение элементу массива с именем ident.Если ни одна из ключевых слов public, protected, или private не определены, принимается  видимость по умолчанию.

Второй синтаксис ("определение макросов") определяет макрос с именем ident. При определении макроса не должно быть пробелов между именем макроса и открывающей скобки, в противном случае он будет рассматриваться как объявление переменной. 

Третий синтаксис ("установка области видимости по умолчанию") устанавливает область видимости по умолчанию для дальнейших переменных и макроопределений в этом файле. Если в скрипте нет объявление области видимости, видимость public принимается по умолчанию.

Примеры

#define MyAppName "My Program"
#define MyAppVer GetFileVersion("MyProg.exe")
#define MyArray[0] 15
#define Multiply(int A, int B = 10) A * B
