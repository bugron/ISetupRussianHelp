Pascal Scripting Extended: TextToShortCut

Прототип:

function TextToShortCut(Text: String): TShortCut;

Описание:
Функция преобразовывает строковое значение комбинации "горячих" клавиш в значение типа TShortCut, которое может быть использовано для переопределения комбинации "горячих" клавиш меню, во время выполнения программы, а также может быть передано в качестве параметра функции NewItem. С помощью функции TextToShortCut можно организовать назначение комбинации "горячих клавиш самим пользователем. Функция TextToShortCut работает более медленно, чем функция ShortCut
Пример:

var

ShortCut: TShortCut;

begin

ShortCut:= TextToShortCut( Edit1.Text);
 MainMenu1.items[0].ShortCut:= ShortCut;

end;
