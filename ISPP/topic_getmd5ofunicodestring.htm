Inno Setup Preprocessor: GetMD5OfUnicodeString

Прототип

str GetMD5OfString(str)

Описание

Возвращает сумму MD5 указанной Unicode строки, как строку.

Использовать только в Unicode версии Inno Setup, иначе компилятор выдаст ошибку.

Пример

#define MD5 GetMD5OfUnicodeString('Test')
// MD5 = '8e06915d5f5d4f8754f51892d884c477'
