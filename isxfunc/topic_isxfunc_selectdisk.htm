Pascal Scripting: SelectDisk

Прототип:

function SelectDisk(const DiskNumber: Integer; const AFilename: String; var Path: String): Boolean;

Описание:

Показывает окно сообщения "Необходимо вставить следующий диск" и возвращает True, если пользователь вводит путь, содержащий файл, который указан в AFileName и нажимает OK. Выбранный путь сохраняется в переменной Path. Текущее значение переменной Path используется как изначально выбранный путь, а DiskNumber используется только в качестве информации для пользователя.
