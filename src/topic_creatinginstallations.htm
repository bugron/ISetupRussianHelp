Создание инсталляторов

Инсталляторы создаются с помощью скриптов, которые представляют собой ASCII или Unicode (UTF-8 с BOM) закодированные текстовые файлы, форматом похожые на INI-файлы (на самом деле писать скрипты не такое уж сложно).

Скрипты имеют расширение ".iss" (сокращение от Inno Setup Script). Скрипт контролирует каждый аспект процесса установки. В скрипте описывается какой файл куда следует установить, какие ярлыки создать и как они будут именоваться, и т.д.

Скрипты обычно редактируются в программе "Inno Setup Compiler" Compiler IDE. После написания скрипта его следует откомпилировать (для этого надо нажать кнопку Запустить или нажать F9). После этого на базе скрипта будет создан готовый к запуску инсталлятор Вашей программы. По умолчанию инсталлятор программы создается в папке Output в папке скрипта.

Чтобы посмотреть, как все это работает, запустите Compiler IDE, нажмите на Файл | Открыть и выберите один из примеров в папке Examples, который находится в директории установки Inno Setup. (Может быть полезно использовать примеры как шаблоны для Ваших скриптов.)
