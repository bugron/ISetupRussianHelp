Inno Setup Preprocessor: GetFileVersion

Прототип

str GetFileVersion(str)

Описание

Функция GetFileVersion получает строковый параметр, который должен быть установлен в имя файла, информацию о версии которого должна быть запрошена. Функция возвращает строку, составленную из четырех десятичных чисел разделенных точками. Если файл не содержит допустимую информацию о версии, функция возвращает пустую строку.

ISPP также имеет функцию GetStringFileInfo, который можно использовать для получения версии файла (используя "FileVersion" или "ProductVersion" как второй параметр). Различие - то, что GetFileVersion берет результат из фиксированного блока информации о версии, в отличие от GetStringFileInfo, который извлекает строку из специфического языкового блока.
