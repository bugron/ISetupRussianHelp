Inno Setup Preprocessor: Pos

Прототип функции

int Pos(str 1, str 2)


Описание

Выполняет поиск подстроки (первый аргумент) в другой строке (второй аргумент) и возвращает целое значение, которое характеризует начальную позицию (по первому символу) искомой подстроки в пределах заданной строки. Если подстрока не найдена, то функция Pos возвращает нулевое значение.
