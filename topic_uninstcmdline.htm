Параметры командной строки деинсталлятора

Программа-деинсталлятор (unins???.exe) поддерживает приведенные ниже параметры командной строки. Они пригодятся системным администраторам и другим программам, вызывающим деинсталлятор.

/SILENT, /VERYSILENT

Если указан этот параметр, деинсталлятор не будет спрашивать у пользователя подтверждения удаления и выдавать сообщение, что удаление завершено. Общие файлы, которые больше не используются, будут удалены автоматически без запроса. Все сообщения о критических ошибках будут отображаться. Если указан параметр '/VERYSILENT', окно прогресса удаления не будет отображено.

Если необходима перезагрузка, команда '/NORESTART' не задана, и указан параметр '/VERYSILENT' , деинсталлятор перезагрузит систему без предупреждения.

/SUPPRESSMSGBOXES

Дает деинсталлятору команду подавлять окна сообщений. Дает эффект только при комбинации с '/SILENT' и '/VERYSILENT'. Для деталей см. '/SUPPRESSMSGBOXES' Параметры командной строки инсталлятора.

/LOG

Указывает деинсталлятору создать log-файл в папке TEMP для подробного отчета о действиях и событиях в секции [UninstallRun], и во время деинсталляции, предпринятых в ходе удаления. Применяется для отслеживания ошибок.

Log-файлу присваивается уникальное имя на базе текущей даты. (Существующие файлы не перезаписываются и не дополняются).

Информация, содержащаяся в log-файле, техническая и поэтому не может быть понята пользователем.

/LOG="filename"

Параметр похож на /LOG, но в отличие от него позволяет указать фиксированный путь/имя файла для log-файла. Если файл с указанным именем уже существует, он будет заменен. Если файл не может быть создан, деинсталлятор прервется и выдаст сообщение об ошибке.

/NORESTART

Указывает деинсталлятору не перезагружать систему, даже если это необходимо.
