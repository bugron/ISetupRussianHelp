Pascal Scripting Extended: Int

Прототип:

function Int(E: Extended): Extended;

Описание:
Функция Int возвращает целочисленную часть числа с плавающей точкой, как число с плавающей точкой.
Пример:

begin
 ShowMessage('Round(12.75) = '+IntToStr(Round(12.75)));
 ShowMessage('Trunc(12.75) = '+IntToStr(Trunc(12.75)));
 ShowMessage(' Int(12.75) = '+FloatToStr(Int(12.75)));
 ShowMessage(' Frac(12.75) = '+FloatToStr(Frac(12.75)));
end;


 Round(12.75) = 13
 Trunc(12.75) = 12
 Int(12.75) = 12
 Frac(12.75) = 0.75
