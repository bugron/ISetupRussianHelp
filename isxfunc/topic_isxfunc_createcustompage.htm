Pascal Scripting: CreateCustomPage

Прототип:

function CreateCustomPage(const AfterID: Integer; const ACaption, AОписание: String): TWizardPage;

Описание:

Создает страницу мастера. Страница пуста по умолчанию, вы должны создать свои собственные элементы управления и разместить их на странице (путем установки их родительских свойств к свойству Surface экземпляра TWizardPage, возвращаемой этой функцией).

Пример можно найти в файле CodeClasses.iss.
