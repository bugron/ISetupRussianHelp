Pascal Scripting: RegisterServer

Прототип:

procedure RegisterServer(const Is64Bit: Boolean; const Filename: String; const FailCriticalErrors: Boolean);

Описание:

Регистрирует DLL/OCX с указанным именем файла. Если Is64Bit является True, DLL/OCX файл будет загружен как 64-разрядный образ и зарегистрирован в 64-разрядном процессе. Если FailCriticalErrors является True, система не покажет каких либо сообщений о критической ошибке. При неудаче вызывает исключение.

Например:
begin
  // Зарегистрировать hhctrl.ocx.
  RegisterServer(Is64BitInstallMode, ExpandConstant('{sys}\hhctrl.ocx'), False);
end;

Поскольку мы указываем Is64BitInstallMode как первый параметр, функция зарегистрирует 64-разрядный OCX файл в 64-разрядном каталоге System, когда инсталлятор работает в 64-разрядном режиме инсталляции. Иначе, функция зарегистрирует 32-разрядный OCX файл в 32-разрядном каталоге System.
