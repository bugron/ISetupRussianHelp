Inno Setup Preprocessor: Расширенная версия компилятора

После установки Inno Setup Preprocessor стандартная версия компилятора Inno Setup (файл ISCC.exe) будет заменена на его расширенную версию, которая предоставляет дополнительные параметры для управления процессом предварительной обработки сценариев Inno Setup.

Синтаксис командной строки: iscc [options] scriptfile.iss

Или посредством стандартного ввода: iscc [options] -

Параметры для эмуляции вызова директивы define или pragma:

/d<name>[=<value>] Задает #define public <name> <value>
/$<letter>(+|-) Задает #pragma option -<letter>(+|-)
/p<letter>(+|-) Задает #pragma parseroption -<letter>(+|-)
/i<paths> Задает #pragma include -<paths>
/{#<string> Задает #pragma inlinestart -<string>
/}<string> Задает #pragma inlineend -<string>
/v<number> Задает #pragma verboselevel -<number>

Другие допустимые параметры:
     /O - устанавливает путь вывода файла (замещает команду OutputDir в сценарии);
     /F - устанавливает выходное имя файла (замещает команду OutputBaseFilename в сценарии);
     /Q[p] - тихая компиляция (будут показаны только сообщения об ошибках, "p" - будет показана информация о процессе выполнения).

Пример: iscc /$c- /pu+ "/dLic=Trial Lic.txt" /iC:\INC;D:\INC "c:\isetup\samples\my script.iss"

При успешной компиляции ISCC возвращает код выхода 0. Если параметры командной строки заданы неверно или произошла внутренняя ошибка, то код возврата 1. В случае неудачной компиляции код возврата 2.
