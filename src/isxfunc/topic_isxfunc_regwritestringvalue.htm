Pascal Scripting: RegWriteStringValue

Прототип:

function RegWriteStringValue(const RootKey: Integer; const SubKeyName, ValueName, Data: String): Boolean;

Описание:

Записывает указанное значение реестра типа REG_SZ. При успешном выполнении возвращает True, иначе False.

Комментарии:

Если уже существует значение типа REG_EXPAND_SZ, то записываемое значение тоже должно быть того же типа. Иначе значение будет записано с типом REG_SZ.

Например:
begin
  RegWriteStringValue(HKEY_CURRENT_USER, 'Software\My Company\My Program',
    'UserName', ExpandConstant('{sysuserinfoname}'));
end;
