[Setup]: AlwaysShowComponentsList

Возможные значения:
yes или no

Значение по умолчанию:
yes
Описание:

При значении yes инсталлятор всегда будет отображать список компонентов. При значении no инсталлятор будет показывать список компонентов, только если пользователь выберет тип установки "Выборочный".
