[Setup]: WizardStyle

Возможные значения:
classic или modern

Значение по умолчанию:
classic
Описание:

Не используется с версии 3.0. Inno Setup 2.х поддерживал другой стиль мастера - "classic". Поддержка стиля "classic" прекратилась в Inno Setup 3.0.

При значении modern, инсталлятор/деинсталлятор будет выглядеть более современно, а значения директив WizardResizable и WizardSizePercent соответственно будут yes и 120,120.

Чтобы добавить полную поддержку WizardResizable и WizardSizePercent для Ваших пользовательских элементов управления и страниц инсталлятора, используйте свойства Anchors и KeepSizeY. Смотрите скрипт CodeClasses.iss для примеров.
