Pascal Scripting Extended: ShortCut

Прототип:

function ShortCut(Key: Word; Shift: TShiftState): TShortCut;

Описание:
Создает клавиатурное сокращение. Первый параметр Key будет завершающим символом в сокращении, преобразованным в Word. В данном случае клавишей является X. Параметр Shift представляет специальную клавишу, которая удерживается нажатой во время нажатия другой клавиши, и объявлен как TShiftState = set of (ssShift,ssAlt,ssCtrl,ssLeft,ssRight,ssMiddle, ssDouble);
где ssShift представляет клавишу <Shift>, ssAlt — клавишу <Alt> и т.д.

Пример:

procedure TForm1.FormCreate(Sender: TObject);
begin
   Cut1.Shortcut := Shortcut(Word('X'),[ssCtrl]);
   Copy1.Shortcut := Shortcut(Word('C'),[ssCtrl]);
   Paste1.Shortcut := Shortcut(Word('T'),[ssCtrl]);

end;
