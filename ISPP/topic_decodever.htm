Inno Setup Preprocessor: DecodeVer

DecodeVer

#define DecodeVer(int Ver, int Digits = 3)


Описание

Выполняет преобразование целочисленного 32-бит значения версии в строковый тип данных, в формат "0.0.0.0". Параметр Digits задает количество элементов в формате версии (Major, Minor, Rev и Build). Если четвертый элемент равен нулю (0), то он будет отсутствовать.

Функция объявлена в файле ISPPBuiltins.iss.
