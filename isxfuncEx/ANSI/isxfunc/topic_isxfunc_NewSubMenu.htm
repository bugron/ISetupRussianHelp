Pascal Scripting Extended: NewSubMenu

Прототип:

function NewSubMenu(const ACaption: String; hCtx: Word; const AName: String; Items: array of TMenuItem; AEnabled: Boolean): TMenuItem;

Описание:
Функция создает, во время выполнения программы, и инициализирует пункт меню, при выборе которого открывается подменю.
Элементы подменю передаются в параметре Items. Параметр ACaption определяет текст создаваемого пункта меню. В параметре AName указывается имя, которое будет идентифицировать подменю в исходном коде программы. В параметре hCtx передается идентификатор контекста справки для данного пункта меню. Параметр AEnabled определяет, будет доступно или заблокировано подменю.
При создании подменю с помощью данной функции, оно может быть добавлено в массив элементов меню и передано в качестве параметра функциям NewMenu, NewPopupMenu или NewSubMenu.
Пример:

var

Items: array[0..2] of TMenuItem;
 Item: TMenuItem;
 Key: TShortCut;

begin

Key:=ShortCut(Word('X'),[ssShift]);

Items[0]:= NewItem('Пункт1', Key, false, true, N1click, 0, 'MenuItem1');

Items[1]:= NewLine;

Items[2]:= NewItem('Пункт2', 0, false, true, N2click, 0, 'MenuItem2');

Item:= NewSubMenu('&Меню1', 0, 'Submenu1', Items, true,);
 NewMenu(Form1, 'Menu1', Item);

end;
