Pascal Scripting: ShowExceptionMessage

Прототип:

procedure ShowExceptionMessage;

Описание:

Выводит на экран сообщение, ассоциированное с текущим исключением с помощью message box. Эта функция должна быть вызвана только изнутри секции except, или функцией, вызванной из секции except.

Комментарии:

Если ведение журнала (logging) включено (через параметр командной строки /LOG или директивой SetupLogging секции [Setup]) сообщение будет выведено на экран и записано в журнале.

Например:
var
  I: Integer;
begin
  I := 1;
  try
    // The following line will raise a "Division by zero" exception
    I := I div 0;
  except
    // Catch the exception, show it, and continue
    ShowExceptionMessage;
  end;
end;
