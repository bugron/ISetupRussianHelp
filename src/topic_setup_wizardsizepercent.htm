[Setup]: WizardSizePercent

Формат:

a,b, где a горизонтальный размер, а b вертикальный размер.

Возможные значения:

кажное значение: от 100 до 150

Значение по умолчанию:

120,120, если WizardStyle=modern, иначе 100,100
Описание:

Позволяет увеличить размер по умолчанию для всех окон инсталлятора/деинсталлятора без увеличения размера шрифта. Например, 120 означает увеличение размера окно на 20%.

Если вы укажете только один размер, то горизонтальный и вертикальный размеры оба примут это значение.

Пример:
WizardSizePercent=120

Чтобы добавить полную поддержку WizardResizable и WizardSizePercent для Ваших пользовательских элементов управления, пользовательских страниц инсталлятора, используйте свойства Anchors и KeepSizeY. Смотрите скрипт CodeClasses.iss для примеров.

Примечание: некоторые окна инсталлятора (например, диалог Выберите язык установки) увеличатся только в размере по горизонтали.
