[Setup]: AppendDefaultGroupName

Возможные значения:
yes или no

Значение по умолчанию:
yes
Описание:

По умолчанию при указании пути на странице Выбор папки меню Пуск с помощью кнопки Обзор... (Browse...) инсталлятор автоматически добавляет в конце имя папки, указанной в директиве DefaultGroupName. Например, если DefaultGroupName=My Program и пользователь выбрал "Accessories", путь пропишется как "Accesories\My Program".

Если значение директивы no, то папка с указанным DefaultGroupName именем в конечном каталоге создаваться не будет. На странице выбора появится кнопка "Создать папку" (Make New Folder).
