[Setup]: WizardResizable

Возможные значения:

yes or no

Значение по умолчанию:

yes если WizardStyle=modern, иначе no
Описание:

При значении yes пользователь сможет изменить размер или развернуть основное окно инсталлятора.

Чтобы добавить полную поддержку WizardResizable и WizardSizePercent для Ваших пользовательских элементов управления, пользовательских страниц инсталлятора и TSetupForm форм, используйте свойства Anchors и KeepSizeY. Смотрите скрипт CodeClasses.iss для примеров.
