[Setup]: DiskClusterSize

Значение по умолчанию:
512 (стандартный размер кластера для гибких дисков)
Описание:

Размер кластера на диске. Компилятору надо это знать, чтобы полностью заполнить каждый кластер.

Директива игнорируется, если разбиение на диски не включено (директива DiskSpanning секции [Setup]).
