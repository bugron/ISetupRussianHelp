Pascal Scripting: RegQueryBinaryValue

Прототип:

function RegQueryBinaryValue(const RootKey: Integer; const SubKeyName, ValueName: String; var ResultStr: AnsiString): Boolean;

Описание:

Запрашивает значение реестра типа REG_BINARY и возвращает данные в переменной ResultStr. При успешном выполнении возвращает True. При значении False ResultStr не меняется.
