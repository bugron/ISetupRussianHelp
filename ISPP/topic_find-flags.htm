Inno Setup Preprocessor: Find flags

Описание

Одно из следующих четырех значений должно быть определено:

FIND_MATCH (0) означает, что строка должна соответствовать строку поиска.

FIND_BEGINS (1) означает, что строка должна запуститься со строки поиска.

FIND_ENDS (2) означает, что строка должна закончиться строкой поиска.

FIND_CONTAINS (3) означает, что строка должна содержать строкe поиска.

Любой из следующих модификаторов может быть объединен с одним из предыдущих значений с помощью оператора порязрядного ИЛИ (|):

FIND_CASESENSITIVE (4) означает, что сравнение должно быть чувствительным к регистру.

FIND_AND (0) означает, что этот критерий (пара параметров в функции Find) должен быть встречен так же как предыдущие критерии.

FIND_OR (8) means that it is allowed that this criterium is tested even if previos criteria were not met.

FIND_NOT (16) означает, что этот критерий не должен быть встречен.

FIND_AND и FIND_OR являются взаимоисключающими. Если оба определены, FIND_OR имеет приоритет.

Специальные флаги:

FIND_TRIM (32) означает, что начальные и конечные пробелы должны быть удалены из линии до тестирования его против всех критериев. Этот флаг может быть использован только в третьем аргументе функции Find. Этот флаг не являются взаимоисключающим ни с одним из вышеупомянутых флагов.
