Pascal Scripting Extended: ExtractTemporaryFileEx

Прототип:

procedure ExtractTemporaryFileEx(const FileName: String; const DestDir: String);

Описание:

Распаковывает заданный файл секции [Files] в указанную папку. Чтобы найти расположение временной папки, используйте ExpandConstant('{tmp}').

Распакованные файлы автоматически удаляются при закрытии инсталлятора.

Будет вызвано исключение, если файл не извлекается успешно, если файл не найден, или если файл существует.

Комментарии:

Используйте Flags: dontcopy в секции [Files], если извлеченный файл не нужно установить.

Если непрерывное сжатие включено, то добавьте такие временные файлы в начало секции [Files]. Для того чтобы извлечь произвольный файл из непрерывно сжатого инсталлятора, он сначала должен разжать все предыдущие файлы (во временной буфер в памяти). Это может привести к существенным задержкам, если ряд других файлов перечислены выше указанного файла в секции [Files].
