Pascal Scripting: ExecAsOriginalUser

Прототип:

function ExecAsOriginalUser(const Filename, Params, WorkingDir: String; const ShowCmd: Integer; const Wait: TExecWait; var ResultCode: Integer): Boolean;

Описание:

Выполняет указанный исполняемый или пакетный файл, используя учетные данные пользователя, запустившего программу установки. Для получения дополнительной информации см. Exec и флаг runasoriginaluser секции [Run].

Комментарии:

Эта функция не поддерживается во время деинсталляции.

В очень исключительных случаях неисправности (например, если процесс установки неожиданно закончился), возможно для этой функции вызвать исключение, вместо простого возвращения значения False.
