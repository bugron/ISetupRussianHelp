Pascal Scripting: RegisterTypeLibrary

Прототип:

procedure RegisterTypeLibrary(const Is64Bit: Boolean; const Filename: String);

Описание:

Регистрирует библиотеку типов с указанным именем файла. Если Is64Bit является True, библиотека типов будет зарегистрирован в 64-разрядном процессе. При неудаче вызывает исключение.

Пример:
begin
  // Зарегистрировать stdole2.tlb.
  RegisterTypeLibrary(Is64BitInstallMode, ExpandConstant('{sys}\stdole2.tlb'));
end;

Поскольку мы указываем Is64BitInstallMode как первый параметр, функция зарегистрирует 64-разрядную библиотеку типов в 64-разрядном каталоге System, когда инсталлятор работает в 64-разрядном режиме инсталляции. Иначе, функция зарегистрирует 32-разрядную библиотеку типов в 32-разрядном каталоге System.
