Секция [Messages]

Секция [Messages] используется для определения текста сообщений в инсталляторе и деинсталляторе. Вы можете не создавать эту секцию в своих скриптах, так как по умолчанию все требуемые параметры данной секции берутся из файла Default.isl, который расположен в папке компилятора (или оттуда, куда указывают параметры секции [Languages]).

Однако отдельные сообщения могут быть перекрыты путем создания секции [Messages] в скрипте. Во-первых, вам следует знать идентификатор (ID) сообщения, которое требуется изменить. Его можно легко отыскать в Default.isl. Например, вы хотите сменить кнопку мастера "&Далее >" на "&Вперед >". ID этого сообщения "ButtonNext", поэтому надо создать вот такую секцию [Messages]:

[Messages]
ButtonNext=&Вперед >

Некоторые сообщения используют аргументы, например %1 и %2. Вы можете изменить их порядок (т.е. поставить %2 перед %1) и при необходимости дублировать (т.е. "%1...%1 %2"). Для того чтобы добавить символ "%", наберите его два раза. "%n" означает перевод каретки.

Если вы хотите перевести все тексты Inno Setup на другой язык, не изменяя при этом Default.isl и не заменяя каждое сообщение в создаваемом скрипте, скопируйте Default.isl и сохраните под другим именем вроде MyTranslation.isl. В любом инсталляторе, в котором вы захотите использовать MyTranslation.isl, нужно будет указать в секции [Languages] параметры, указывающие на этот файл.

В случаях, когда в секции [Languages] указано несколько параметров, задание в секции [Messages] параметров (в противоположность .isl файлу) по умолчанию заменит сообщение для всех языков. Для применения параметра секции [Messages] только для одного языка, допишите префикс перед именем языка. Например:

en.ButtonNext=&Forward >

Сообщения специального назначения:

Сообщение BeveledLabel можно использовать, чтобы отобразить строку текста в левом нижнем углу окна инсталлятора и деинсталлятора. Например:

[Messages]
BeveledLabel=Inno Setup

Сообщение HelpTextNote можно использовать для указания одного или более строк текста, который будет показываться, если инсталлятор запустили с параметром командной строки /HELP. Например:

  [Messages]
  HelpTextNote=/PORTABLE=1%nEnable portable mode.

Сообщения специального назначения по умолчанию не имеют значений и если Вы хотите их использовать, убедитесь что установили их значения для всех языков в Вашем главном скрипте.
