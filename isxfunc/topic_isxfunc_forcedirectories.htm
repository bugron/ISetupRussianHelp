Pascal Scripting: ForceDirectories

Прототип:

function ForceDirectories(Dir: string): Boolean;

Описание:

Создает сразу все папки по указанному пути. Если первые папки в пути существуют, а несколько последних нет, ForceDirectories создает только те, которые не существуют. При успешном выполнении возвращает True, в противном случае False.
