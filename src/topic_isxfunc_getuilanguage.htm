Pascal Scripting: GetUILanguage

Прототип:

function GetUILanguage: Integer;

Описание:

Возвращает идентификатор языка интерфейса (LANGID) текущего пользователя. Возвращает 0, если функция потерпит неудачу (что маловероятно).

Комментарии:

Подробнее: Список национальных идентификаторов и языковых групп в MSDN.

Например:
begin
 if GetUILanguage = $0409 then
 begin
 // UI language is English (United States)
 end;

 // You can use "and $3FF" to extract the primary language identifier
 if GetUILanguage and $3FF = $09 then
 begin
 // Matches any variant of English
 end;
end;
