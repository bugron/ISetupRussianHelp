» Inno Setup Faq. How to close an application posting a QUIT message


[Setup]
AppName=Killer
AppVerName=Killer
Uninstallable=false
UpdateUninstallLogAppName=false
DisableDirPage=true
DisableProgramGroupPage=true
DefaultDirName={pf}\Drvtype
DisableStartupPrompt=true

[Code]
function InitializeSetup: Boolean;
var
   WindowNumber, closeWindow:integer;
begin
   closeWindow:=16; ///CloseValue
   ///WindowNumber:=FindWindowByWindowName('test'); ///Find Number by Name
   WindowNumber:=FindWindowByClassName('XLMAIN');
   PostMessage(WindowNumber,closeWindow,0,0); ///close send to Application-Window
   ///next step for installation ;-)
   result:=true;
end;
