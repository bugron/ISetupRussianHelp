Pascal Scripting: RegQueryStringValue

Прототип:

function RegQueryStringValue(const RootKey: Integer; const SubKeyName, ValueName: String; var ResultStr: String): Boolean;

Описание:

Запрашивает значение реестра типа REG_SZ или REG_EXPAND_SZ и возвращает данные в переменной ResultStr. При успешном выполнении возвращает True. При значении False ResultStr не меняется.

Например:
var
  Country: String;
begin
  if RegQueryStringValue(HKEY_CURRENT_USER, 'Control Panel\International',
     'sCountry', Country) then
  begin
    // Successfully read the value
    MsgBox('Your country: ' + Country, mbInformation, MB_OK);
  end;
end;
