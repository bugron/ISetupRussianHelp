Pascal Scripting: DLLGetLastError

Прототип:

function DLLGetLastError(): Longint;

Описание:

Возвращает значение кода последней ошибки, возникшей сразу после вашего последнего вызова DLL функции. Полезна после вызова функции Windows API (если функция устанавливает код последней ошибки).

Комментарии:

Рекомендуется использовать эту функцию вместо прямого вызова WinAPI функции GetLastError, так как инсталлятор/деинсталлятор делает API вызовы самостоятельно, так что последний код ошибки может быть перезаписан в любое время.

Ссылка на коды системных ошибок в MSDN.

Например:
function MessageBox(hWnd: Integer; lpText, lpCaption: AnsiString; uType: Cardinal): Integer; external 'MessageBoxA@user32.dll stdcall';

...

begin
  if MessageBox(-1, '', '', -1) = 0 then
    MsgBox(SysErrorMessage(DLLGetLastError), mbError, mb_Ok);
