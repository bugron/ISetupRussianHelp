Pascal Scripting: Copy

Прототип:

function Copy(S: String; Index, Count: Integer): String;

Описание:

Возвращает строку, содержащую Count символов, начинающиеся с позиции S[Index].
Если Index больше чем длина S, функция Copy возвращает пустую строку.
Если Count определяет больше символов, чем доступно, возвращаются символы с позиции S[Index] до конца строки S.
