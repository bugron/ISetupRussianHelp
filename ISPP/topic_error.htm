Inno Setup Preprocessor: #error

Синтатсис

error-directive: error <text>

Описание

Директива указывает компилятору Inno Setup выдавать сообщение об ошибке с указанным текстом. В отличие от pragma error, текст в директиве error не разбирается, поэтому его рекомендуется использовать эту директиву, а не pragma когда это возможно, чтобы избежать возможных ошибок синтаксиса, которые могут скрывать реальные ошибки. 

Примеры

#if VER < 0x04000000
  #error A more recent version of Inno Setup is required to compile this script
#endif

См. также:

pragma, if.
