Inno Setup Preprocessor: #file

Синтатсис

file-directive: file <expr>

Описание

Заменяет директиву с именем временного файла, содержащего перевод указанного файла. После завершения компиляции временной файл автоматически удаляется.

Включая используемый файл, эта директива создает новый независимый экземпляр препроцессора, передавая ему и опции в реальном времени, и все объявленные идентификаторы. Если включенный файл изменяет опции в некотором роде, они не распространяются назад.

При использовании этой директивы в параметре Source  секции [Files] также определите параметр DestName, иначе файл не будет установлен с исходным именем.

Данная директива может использоваться только встроенно.

Примеры

[Setup]
LicenseFile={#file "mylic.txt"}
