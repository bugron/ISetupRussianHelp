Inno Setup Preprocessor: #sub, #endsub

Синтатсис

sub-directive: sub <ident>
endsub-directive: endsub

Описание

Директивы sub и endsub используются, чтобы объявить определяемую пользователем процедуру, которая является частью скрипта, который может быть включен позже однажды или несколько раз. You make think of a user defined procedure as begin similar to an external file, and a call to a user defined procedure as being similar to inclusion of an external file, except that procedures may also be called from within expressions. Please note that it is strongly not recommended to call procedures which emit several lines to translation from within compound expressions or directives.

Процедура вызывается, просто указанием ее идентификатора, с помощью которого она была объявлена.

Выполнение процедуры начинается только после ее вызова, так что если существуют какие-либо ошибки в ее теле, они выскочат только тогда, когда процедура вызывается.

Примеры

#sub AddFile
  #if Copy(FileName, 1, 1) == "A"
    Source: {#FileName}; DestDir: {app}\A
  #else
    Source: {#FileName}; DestDir: {app}
  #endif
#endsub
