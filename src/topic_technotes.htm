Прочие замечания

  • Для упрощения автоматического обновления вашего приложения, сначала каким-либо образом сделайте так, чтобы приложение обнаруживало новую версию Setup.exe и находило или загружало новую версию. Затем, для автоматического обновления, запустите Setup.exe из приложения, используя, например, следующую командную строку: 
    /SP- /silent /noicons "/dir=expand:{autopf}\My Program"

    После запуска Setup.exe выйдите из приложения как можно быстрее. Для избегания проблем с обновлением .exe файла, инсталлятор может автоматически повторять попытки несколько раз.

    По желанию можно использовать флаги skipifsilent и skipifnotsilent и сделать так, чтобы приложение имело параметр '/updated', например, чтобы отображать красивое окно сообщения для информирования пользователя о том, что обновление выполнено.

  • Инсталлятор самого Inno Setup поддерживает дополнительный параметр командной строки /PORTABLE=1 для включения портативного режима при котором инсталлятор при установке не создает ни деинсталлятор, ни запись в "Установке и удалении программ" панели управления.
  • Резервный веб-сайт Inno Setup можно найти по адресу www.innosetup.nl. Обычно эта ссылка просто перенаправит на основной веб-сайт по адресу jrsoftware.org, но в случае чрезвычайной ситуации это перенаправление будет отключено.
