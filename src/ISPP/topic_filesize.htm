Inno Setup Preprocessor: FileSize

Прототип функции

int FileSize(str)


Описание

Возвращает размер файла в байтах. Если файл не существует, то возвращается значение -1 (минус один). Внимание, ISPP поддерживает только 32-бит целые числа. Для файлов с размерами более 2 Гб (и менее 4 Гб) возвращаемое значение всегда будет отрицательным.
