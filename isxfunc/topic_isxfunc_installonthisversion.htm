Pascal Scripting: InstallOnThisVersion

Прототип:

function InstallOnThisVersion(const MinVersion, OnlyBelowVersion: String): Boolean;

Описание:

Эта функция устарела. Возвращает True, if an entry with the specified MinVersion and OnlyBelowVersion parameters should be installed. При передаче недопустимой строки генерируется исключение.

Функция предоставлена только для обратной совместимости и в будущем может быть удалена. В новых скриптах используйте GetWindowsVersion или GetWindowsVersionEx.

Комментарии:

До Inno Setup 5.5.0, данная функция возвращала irInstall, а не логическое значение True. irInstall в настоящее время определяется как псевдоним для True.

Например:
// Old method, deprecated
function IsWindowsXPOrLater: Boolean;
begin
  Result := InstallOnThisVersion('0,5.1', '0,0');
end;

// New method
function IsWindowsXPOrLater: Boolean;
begin
  Result := (GetWindowsVersion >= $05010000);
end;
