Pascal Scripting: GetOpenFileNameMulti

Прототип:

function GetOpenFileNameMulti(const Prompt: String; var FileNameList: TStrings; const InitialDirectory, Filter, DefaultExtension: String): Boolean;

Описание:

Показывает диалоговое окно, которое позволяет пользователю выбрать один или несколько файлов. Возвращает True, если пользователь выбрал файл, иначе False. Список выбранных файлов сохраняется в FileNameList.

Комментарии:

Пример значения переменной Filter: 'Text files (*.txt)|*.txt|All files (*.*)|*.*'

Например:
var
  FileNameList: TStrings;
begin
  // Create the list
  FileNameList := TStringList.Create;
  try
    if GetOpenFileNameMulti('', FileNameList, '',
       'Text Documents (*.txt)|*.txt|All Files|*.*', 'txt') then
    begin
      // Successful; user clicked OK
      // FileNameList contains the selected filename(s)
    end;
  finally
    FileNameList.Free;
  end;
end;
