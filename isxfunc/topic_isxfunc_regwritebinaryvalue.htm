Pascal Scripting: RegWriteBinaryValue

Прототип:

function RegWriteBinaryValue(const RootKey: Integer; const SubKeyName, ValueName: String; const Data: AnsiString): Boolean;

Описание:

Записывает указанное значение реестра типа REG_BINARY. При успешном выполнении возвращает True, иначе False.

Например:
begin
  RegWriteBinaryValue(HKEY_CURRENT_USER, 'Software\My Company\My Program',
    'BinaryTest', 'Whatever' + #1#2#3#4);
end;
