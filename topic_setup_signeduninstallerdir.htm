[Setup]: SignedUninstallerDir

Возможные значения:
OutputDir
Описание:

Определяет каталог, в который сохранены файлы подписанного деинсталлятора. По умолчанию, такие файлы хранятся в OutputDir.

Отдельные файлы скрипта могут совместно использовать один и тот же SignedUninstallerDir. Устанавливая общий каталог, содержащий подписанные файлы деинсталлятора, Вы не должны подписывать деинсталлятор каждый раз, когда Вы компилируете новый скрипт с отличной от значения по умолчанию OutputDir.

Если SignedUninstallerDir не будет полностью определенным путем, то он будет обработан как являющийся относительно SourceDir. Если задать SignedUninstallerDir значение ".", файлы поместятся в исходный каталог.

Например:
SignedUninstallerDir=c:\signeduninstallers
