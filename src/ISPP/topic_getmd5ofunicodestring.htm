Inno Setup Preprocessor: GetMD5OfUnicodeString

Прототип функции

str GetMD5OfString(str)


Описание

Возвращает строковое значение контрольной суммы MD5 в кодировке Unicode для указанной строки.

Внимание, компиляцию сценария, в котором используется вызов этой функции, необходимо выполнять в Unicode версии Inno Setup, иначе будет получена внутренняя ошибка.


Пример

    // Объявляем переменную MD5 и присваиваем ей значение
  // контрольной суммы строки "Test", которое вернёт функция.
  #define MD5 GetMD5OfUnicodeString('Test')
  // MD5 = '8e06915d5f5d4f8754f51892d884c477'
