Pascal Scripting: DownloadTemporaryFileSize

Прототип:

function DownloadTemporaryFileSize(const Url): Int64;

Описание:

Возвращает размер файла с указанного URL не скачивая этот файл. Возврващет -1, если сервер не предоставляет размер файла.

При ошибке будет порождено исключение.

См. DownloadTemporaryFile для других рекомендаций.
