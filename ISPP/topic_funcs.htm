Inno Setup Preprocessor: Функции

ISPP содержит ряд встроенных функций, которые можно использовать для выполнения каких-либо операций во время компиляции и/или изменения сценария. Например, сначала получим версию исполняемого файла программы, для которой создан сценарий установки, а затем, полученным значением, которое вернёт функция, изменим значение директивы Inno Setup.

   ; Объявляем переменную с именем MyAppVer и присваиваем ей значение,
  ; которое вернёт функция GetFileVersion() (получение версии ЕХЕ-файла).
  #define MyAppVer GetFileVersion(AddBackslash(SourcePath) + "MyProg.exe")

  ; Раздел параметров инсталляции
  [Setup]
  ; Используем значение переменной MyAppVer в определении директивы AppVerName.
  AppVerName=MyProg version {#MyAppVer}
