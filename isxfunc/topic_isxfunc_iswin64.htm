Pascal Scripting: IsWin64

Прототип:

function IsWin64: Boolean;

Описание:

Возвращает True, если система является 64-разрядной версией ОС Windows, обеспечивающей поддержку всю API, которое требуется Inno Setup для выполнения 64-разрядных задач установки. Если было возвращено False, Вы не сможете использовать любой из 64-разрядных функции Inno Setup-а.

Комментарии:

Функция всегда возвратит True на 64-разрядных версиях Windows.

Чтобы функция возвращала True на Itanium версиях Windows, требуется, чтобы ОС была Windows Server 2003 SP1 или более поздней версии. Старые версии не имеют требуемых Inno Setup API функций (например, RegDeleteKeyEx).

Например:
begin
  // Check IsWin64 before using a 64-bit-only feature to
  // avoid an exception when running on 32-bit Windows.
  if IsWin64 then
  begin
    MsgBox('64-bit program files reside in: ' +
      ExpandConstant('{pf64}'), mbInformation, MB_OK);
  end;
end;
