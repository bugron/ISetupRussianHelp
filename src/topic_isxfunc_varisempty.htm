Pascal Scripting: VarIsEmpty

Прототип:

function VarIsEmpty(const V: Variant): Boolean;

Описание:

Не Unicode Inno Setup: Возвращает True, если значение указанной переменной не определено (undefined). Unicode Inno Setup: Возвращает True, если указанной переменной не присвоено значение.
