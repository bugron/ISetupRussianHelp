Inno Setup Preprocessor: FindNext

Прототип

int FindNext(int)

Описание

FindNext возвращает следующую запись, которая соответствует имени и атрибутам определенных в предыдущем вызове FindFirst. Параметр должен быть дескриптором поиска, возвращенным FindFirst. Возвращаемое значение является ненулевым, если функция была успешна выполнена.
