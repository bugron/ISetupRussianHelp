Pascal Scripting: ExtractRelativePath

Прототип:

function ExtractRelativePath(const BaseName, DestName: String): String;

Описание:

Конвертирует полный путь в относительный. Параметр DestName указывает имя файла (включая путь) для конвертации. BaseName - это полное имя папки, к которому возвращаемый путь является относительным. BaseName может или не может включать в себя имя файла, но она должна содержать окончательный разделитель пути.

Например:
var
  S: String;
begin
  S := ExtractRelativePath('c:\windows\system32\', 'c:\autoexec.bat');
  // S = ..\..\autoexec.bat
end;
