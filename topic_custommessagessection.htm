Секция [CustomMessages]

В секции [CustomMessages] задаются индивидуальные значения сообщений для констант {cm:...}. Для информации обратитесь к разделу Константы.

Пример задания с описанием, взятым из секции [CustomMessages] с использованием константы {cm:...}:

[CustomMessages]
CreateDesktopIcon=Create a &desktop icon

[Tasks]
Name: desktopicon; Description: "{cm:CreateDesktopIcon}"

Сообщения могут содержать константы с %1 до %9. Вы можете изменить порядок этих символов (т.е. поставить %2 перед %1) и при необходимости дублировать их (т.е. "%1...%1 %2"). Для того чтобы добавить символ "%", наберите его два раза. "%n" означает перевод каретки.

В случаях, когда в секции [Languages] указаны несколько параметров, заданные в секции [CustomMessages] параметры вашего скрипта (в противоположность .isl файлу) по умолчанию заменят сообщения для всех языков. Для применения параметра секции [CustomMessages] только для одного языка, допишите префикс перед именем языка.

nl.CreateDesktopIcon=Maak een snelkoppeling op het &bureaublad

В настоящее время .isl-файлы для всех языков, поддерживаемых Inno Setup, имеют следующие стандартные сообщения, описанные и переведенные для каждого языка (показаны значения для английского языка):

NameAndVersion=%1 version %2
AdditionalIcons=Additional icons:
CreateDesktopIcon=Create a &desktop icon
CreateQuickLaunchIcon=Create a &Quick Launch icon
ProgramOnTheWeb=%1 on the Web
UninstallProgram=Uninstall %1
LaunchProgram=Launch %1
AssocFileExtension=&Associate %1 with the %2 file extension
AssocingFileExtension=Associating %1 with the %2 file extension...

Можете использовать приведенные стандартные сообщения в своих скриптах. Пример, который использует UninstallProgram:

[Icons]
Name: "{group}\{cm:UninstallProgram,My Program}"; Filename: "{uninstallexe}"
