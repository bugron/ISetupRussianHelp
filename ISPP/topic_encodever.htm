Inno Setup Preprocessor: EncodeVer

EncodeVer

#define EncodeVer(int Major, int Minor, int Revision = 0, int Build = -1)

Описание

Возвращает переданные четыре элемента, закодированные в 32-битное целое число (8 бит для каждого из элементов, т.е. элементы должны имеет значение 0...255).

Объявлен в файле ISPPBuiltins.iss.
