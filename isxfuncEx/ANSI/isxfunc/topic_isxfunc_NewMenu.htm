Pascal Scripting Extended: NewMenu

Прототип:

function NewMenu(Owner: TComponent; const AName: String; Items: array of TMenuItem): TMainMenu;

Описание:
Функция создает, во время выполнения программы, и инициализирует меню верхнего уровня.
Параметр Items определяет массив элементов меню, параметр AName - имя которое будет идентифицировать меню в тексте программы. Параметр Owner определяет форму, которой будет принадлежать создаваемое меню. Массив элементов меню можно заполнить с помощью функции NewItem.
Пример:

var

Items: array[0..2] of TMenuItem;
 Item: TMenuItem;
 Key: TShortCut;

begin

Key:=ShortCut(Word('X'),[ssShift]);

Items[0]:= NewItem('Пункт1', Key, false, true, N1click, 0, 'MenuItem1');

Items[1]:= NewLine;

Items[2]:= NewItem('Пункт2', 0, false, true, N2click, 0, 'MenuItem2');

Item:= NewSubMenu('&Меню1', 0, 'Submenu1', Items, true,);
 NewMenu(Form1, 'Menu1', Item);

end;
