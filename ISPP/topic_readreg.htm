Inno Setup Preprocessor: ReadReg

Прототип

any ReadReg(int 1, str 2, str? 3, any? 4)

Описание

Получает значение указанного ключа 32-х битного или 64-х битного системного реестра. Первый параметр - корневой ключ, такой как HKEY_LOCAL_MACHINE (32-х битный) или HKEY_LOCAL_MACHINE_64 (64-х битный). В данной функции можно использовать константы, объявленные в файле BUILTINS.ISS. Второй параметр определяет подключ. Третий параметр определяет имя значения, если этот параметр не указан, принимается значение по умолчанию. Последний дополнительный параметр может использоваться, чтобы определить значение по умолчанию, которое будет возвращено при отказе.

Отметьте, что эта функция может возвращать значение любого типа в зависимости от типа значения в реестре.
