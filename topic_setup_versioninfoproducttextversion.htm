[Setup]: VersionInfoProductTextVersion

Значение по умолчанию:
Если установлено - VersionInfoProductVersion, иначе AppVersion, если не содержит констант (иначе будет пустая строка), иначе VersionInfoTextVersion
Описание:

Задает текстовый файл с описанием версии программы.

Игнорируется, если UseSetupLdr=no.
