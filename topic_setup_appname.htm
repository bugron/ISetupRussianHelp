[Setup]: AppName

Описание:

Эта обязательная директива определяет название устанавливаемого приложения. Здесь не следует указывать номер версии - его указывают в директивах AppVersion и/или AppVerName. При классическом стиле, текст из AppName отображается в верхнем левом углу главного окна установки и окне мастера. Значение может содержать константы.

Значение этой директивы также используется в качестве значения по умолчанию для AppId, VersionInfoDescription, и VersionInfoProductName, если те не определены.

Например:
AppName=My Program
