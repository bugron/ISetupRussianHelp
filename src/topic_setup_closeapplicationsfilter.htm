[Setup]: CloseApplicationsFilter

Возможные значения:
Список расширений файлов, разделенных запятыми

Значение по умолчанию:
*.exe,*.dll,*.chm
Описание:

Ограничивает записи [Files] и [InstallDelete], подвергаемые проверке в использовании перед установкой. Будут проверены только те файлы, которые соответствуют одному из подстановочных знаков.

Установив значение директивы в *.*, можно обеспечить лучшую проверку за счет скорости.

См. также:
CloseApplications
RestartApplications
