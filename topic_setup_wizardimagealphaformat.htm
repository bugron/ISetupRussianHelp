[Setup]: WizardImageAlphaFormat

Возможные значения:

none, defined, premultiplied

Значение по умолчанию:

none
Описание:

При значении none, по умолчанию, любая картинка мастера, которая является 32-х битным растровым изображением, не должна иметь альфа-канал.

При значении premultiplied, любая картинка мастера, которая является 32-х битным растровым изображением, должна иметь альфа-канал с предварительным умножением значения альфа-канала со значениями красного, зеленого и синего каналов.

При значении defined, любая картинка мастера, которая является 32-х битным растровым изображением, должна иметь альфа-канал без предварительного умножения значения альфа-канала со значениями красного, зеленого и синего каналов.

Данная директива действует только для тех картинок мастера, которые являются 32-х битными растровыми изображениями.
