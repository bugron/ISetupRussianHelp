Секция [INI]

Это не обязательная секция. Она определяет ключи и значения, создаваемые инсталлятором в .INI файлах после копирования файлов.

Пример секции [INI]:

[INI]
Filename: "MyProg.ini"; Section: "InstallSettings"; Flags: uninsdeletesection
Filename: "MyProg.ini"; Section: "InstallSettings"; Key: "InstallPath"; String: "{app}"

Список поддерживаемых параметров:

Filename (Обязательно)

Имя .INI файла, который инсталлятор должен создать или модифицировать, может содержать константы. Если этот параметр пустой, запись будет производиться в файле WIN.INI в системной папке Windows.

Например:
Filename: "{app}\MyProg.ini"
Section (Обязательно)

Имя секции, в котором нужно сделать запись, может содержать константы.

Например:
Section: "Settings"
Key

Имя ключа, в который нужно записать данные, может содержать константы. Если этот параметр не указан или пустой, ключ не создается.

Например:
Key: "Version"
String

Значение, которое нужно прописать в ключе, может содержать константы. Если этот параметр не указан или пустой, ключ не создается

Например:
String: "1.0"
Flags

Параметры для установки дополнительных опций. Разделяются пробелами. Поддерживаемые значения:

createkeyifdoesntexist

Записывать значение в ключ, только если ключ не существует. Если этот флаг не указан, значение будет записано в ключ независимо от его существования.

uninsdeleteentry

Удалить запись при удалении программы. Может комбинироваться с флагом uninsdeletesectionifempty.

uninsdeletesection

После удаления программы удалить секцию с параметром, в котором он находится. Не рекомендуется использовать этот флаг для секции, которой пользуется Windows (например, некоторые секции в WIN.INI). Следует использовать этот флаг только в секциях вашего приложения.

uninsdeletesectionifempty

То же, что и uninsdeletesection, но удаляет секцию, только если в ней не осталось ключей. Может комбинироваться с флагом uninsdeleteentry.

Например:
Flags: uninsdeleteentry
