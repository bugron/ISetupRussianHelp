Inno Setup Preprocessor: Find

Прототип

int Find(int 1, str 2, int? 3, str? 4, int? 5, str? 6, int? 7)

Описание

Find функция предназначен для использования с директивой insert. Функция возвращает индекс строки в переводе в зависимости от указанных критериев.

Первый параметр обозначает индекс строки, с которой начнется поиск, обычно он равен нулю.

Второй, четвертый и шестой параметры должны определить строку (и) для поиска в пределах каждой из строк. Второй параметр должен быть определен, а четвертый и шестой могут быть не заданы.

Третий, пятый, и седьмой параметры должны определить поисковые флаги для каждой строки, это означает, что третий параметр определяет флаги для второго, пятое для четвертого, и седьмой для шестого.

Если какой-либо 'флаг' не задан, но строковый параметр, предшествующий ему, определен, принимается FIND_MATCH | FIND_AND (то есть 0).

Значения третьей, пятой, и седьмой параметров функции Find объявлены в файле BUILTINS.ISS. См. тему Find flags для описания каждого из значений.
