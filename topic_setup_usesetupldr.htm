[Setup]: UseSetupLdr

Возможные значения:
yes или no

Значение по умолчанию:
yes
Описание:

Определяет тип установочной комплектации. При значении yes, все нужные файлы (и устанавливаемые в том числе) сжимаются в SETUP.EXE (вы сможете его переименовать). При значении no, создается как минимум три файла: SETUP.EXE, SETUP.0 и SETUP-1.BIN. Используется значение no только в случае отключения компонентов.

Обратите внимание: UseSetupLdr=no не используется при разбиении на диски (DiskSpanning=yes). При UseSetupLdr=yes программа установки копируется и запускается из временной папки TEMP. Если же UseSetupLdr=no, это может вызвать ошибки, если Windows попытается найти файл setup.exe на диске и не найдет его, потому что будет вставлен другой диск.

Также UseSetupLdr=no не используется, чтобы избежать задержек запуска проверки цифровой подписи большого инсталлятора, вместо этого используйте разбиение на диски. См. SignTool для получения дополнительной информации. Если UseSetupLdr=no, то цифровая подпись для деинсталлятора будет недопустима.
