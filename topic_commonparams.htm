Общие параметры

Есть три дополнительные параметры, которые могут использоваться во всех секциях, содержащих параметры. Это:

Languages

Список поддерживаемых языков, разделенных пробелами, которые указывают инсталлятору, к какому языку относится запись. Если пользователь выберет язык, указанный в этом списке, запись будет обработан (например, установится файл).

Запись без указанного параметра Languages всегда обрабатывается, если только другой параметр этого не запретит.

Кроме списка, можно использовать логические выражения. Для справки обратитесь к параметрам секций [Components] и [Tasks].

Например:
Languages: en nl
MinVersion

Минимальная версия Windows, необходимая для обработки записи. Если значение равно "0", значит запись не будет обработана. Номера сборки и/или пакета обновления могут содержаться в значении. Перезаписывет любое значение директивы MinVersion секции[Setup].

Запись, для которой не указан MinVersion, обрабатывается всегда, если только другой параметр этого не запретит.

Для совместимости с предыдущими версиями Inno Setup, отдельные номера Windows 95/98/Me и номера версий Windows NT могут быть определены и разделены запятой. Например: 4.1,5.0. Номер версии Windows 95/98/Me (первое число) не используется, поскольку Inno Setup больше не поддерживает системы Windows 95/98/Me.

Например:
MinVersion: 5.1
OnlyBelowVersion

Параметр, обратный MinVersion. Указывает минимальную версию Windows, чтобы запись не была обработана. Например, если Вы ввели 6.0 и инсталлятор запущен в системах Windows 2000 или XP, то запись будет обработана, но если система пользователя будет Windows Vista (версия 6.0) или выше, запись будет не обработана. Если значение равно "0", значит, запись будет обработана на любой платформе Windows. Номера сборки и/или пакета обновления могут содержаться в значении. Перезаписывет любое значение директивы OnlyBelowVersion секции [Setup].

Запись, для которой не указанOnlyBelowVersion, обрабатывается всегда, если только другой параметр этого не запретит.

Для совместимости с предыдущими версиями Inno Setup, отдельные номера Windows 95/98/Me и номера версий Windows NT могут быть определены и разделены запятой. Например: 4.1,5.0. Номер версии Windows 95/98/Me (первое число) не используется, поскольку Inno Setup больше не поддерживает системы Windows 95/98/Me.

Например:
OnlyBelowVersion: 6.0
