Inno Setup Preprocessor: Exec

Прототип

int Exec(str 1, str? 2, str? 3, int? 4, int? 5)

Описание

Выполняет указанный исполняемый файл.

Первый аргумент определяет имя файла модуля для выполнения.

Второй аргумент может быть использован для указания командной строки для выполнения.

Третий аргумент может быть использован для указания рабочего каталога процесса.

Четвертый аргумент должен быть установлен в ноль, если вы не хотите ждать завершения процесса, и отлично от нуля в противном случае. По умолчанию предполагается значение отличное от нуля. 

Пятый аргумент может быть любой из констант SW_ * обычно определенных в  BUILTINS.ISS file. Для процессов GUI, она определяет значение по умолчанию во время первого вызова ShowWindow. По умолчанию предполагается SW_SHOWNORMAL (т.е. 1). 

Если четвертый аргумент пропущен или не равно нулю, то функция возвращает код завершения процесса. В противном случае, результат функции указывает, является ли процесс успешно запущенным (при успехе значение отличное от нуля).
