Inno Setup Preprocessor: DecodeVer

DecodeVer

#define DecodeVer(int Ver, int Digits = 3)

Описание

Возвращает декодированную строку переданного 32-битного закодированного целого числа. Параметр Digits указывает сколько элементов показать (если четвертый элемент будет 0, то его не покажут так или иначе).

Объявлен в файле ISPPBuiltins.iss.
