Inno Setup Preprocessor: GetFileVersion

Прототип функции

str GetFileVersion(str)


Описание

Возвращает строковое значение версии файла. В качестве аргумента функция принимает имя файла, для которого необходимо определить версию. Формат строки с версией представляет собой четыре десятичных числа, разделённых точками (0.0.0.0). Если указанный файл не существует или не найден, то функция возвращает пустую строку.

В ISPP также имеется функция GetStringFileInfo, которая также позволяет получить информацию о версии файла (если в качестве второго параметра использовать "FileVersion" или "ProductVersion"). Разница между этими двумя функциями заключается в том, что GetFileVersion получает данные о версии из блока ресурсов, а GetStringFileInfo извлекает строку из определённого языкового блока.
