[Setup]: LicenseFile

Описание:

Имя файла с текстом лицензии в формате .txt или .rtf, который отображается перед выбором папки. Этот файл должен быть расположен в исходной папке скрипта, если только для него не указан полный путь или нет префикса "compiler:" - в этом случае компилятор будет искать указанный файл в своем каталоге.

Если пользователь выберет язык, для которого существует параметр LicenseFile, эта директива будет игнорирована. Для справки обратитесь к описанию секции [Languages].

Например:
LicenseFile=license.txt
