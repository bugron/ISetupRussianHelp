Pascal Scripting: DelTree

Прототип:

function DelTree(const Path: String; const IsDir, DeleteFiles, DeleteSubdirsAlso: Boolean): Boolean;

Описание:

Удаляет указанную папку, если IsDir является True, или файлы/папки, соответствующие подстановочному знаку, если IsDir является False. При успешном выполнении возвращает True.

Если DeleteFiles является True, файлы в указанной папке будут удалены, если IsDir является True, или будут удалены файлы, соответствующие указанному подстановочному знаку (включая скрытые, системные и файлы только для чтения), если IsDir является False.

DeleteFiles и DeleteSubdirsAlso являются True, будут удалены также вложенные папки (с их содержимым).

Комментарии:

Эта функция удалит папки, которые являются точками повторной обработки, но не будет рекурсивно удалять файлы/папки внутри них.

Например:
begin
  // Delete the directory C:\Test and everything inside it
  DelTree('C:\Test', True, True, True);

  // Delete files matching C:\Test\*.tmp
  DelTree('C:\Test\*.tmp', False, True, False);

  // Delete all files and directories inside C:\Test
  // but leave the directory itself
  DelTree('C:\Test\*', False, True, True);
end;
