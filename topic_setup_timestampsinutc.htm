[Setup]: TimeStampsInUTC

Возможные значения:
yes или no

Значение по умолчанию:
no
Описание:

По умолчанию временные отметки файлов секции [Files], сохраняются и отображаются как локальные. Это значит, что если отдельный файл имеет временную отметку 01:00 по локальному времени во время компиляции, инсталлятор установит файл с отметкой 01:00, игнорируя временную зону, в которой находится пользователь или при действии DST.

При значении yes временные отметки будут сохранены и отображены в UTC - временном формате Win32 и NTFS. В этом режиме файл с временной отметкой 01:00 по локальному времени в Нью-Йорке получит временную отметку 06:00 по локальному времени в Лондоне.
