Pascal Scripting: CreateOutputProgressPage

Прототип:

function CreateOutputProgressPage(const ACaption, AОписание: String): TOutputProgressWizardPage;

Описание:

Создает страницу мастера, которая содержит статический текст и индикатор хода выполнения (который скрыт по умолчанию).

В отличие от других типов страниц мастера, данные страницы не показываются в обычной последовательности страниц (обратите внимание, что нет параметра AfterID). Страница прогресса можно показать только программно, с помощью метода Show.

Комментарии:

Вызовите метод Show, чтобы показать страницу. Чтобы вернуться к предыдущей странице, вызовите метод Hide.

Всегда размещайте вызов Hide внутри finally, части конструкции try..finally, как показано в CodeDlg.iss. Если не вызвать Hide, инсталлятор "застрянет" на странице прогресса.

Чтобы указать текст на странице, вызовите метод SetText. SetText принимает два строковых параметра: используйте первый, чтобы указать пользователю что вы делаете и второй, чтобы показать имя файла, или папки. Любой из этих двух параметров может быть пустым.

Чтобы показать или обновить индикатор хода выполнения, вызовите метод SetProgress. SetProgress принимает два целочисленных параметра: первых указывает позицию индикатора (обычно 0), а второй определяет максимально возможную позицию. Если второй параметр является 0, индикатор выполнения будет скрыт.
