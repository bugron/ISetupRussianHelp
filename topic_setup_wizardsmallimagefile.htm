[Setup]: WizardSmallImageFile

Значение по умолчанию:
compiler:WIZMODERNSMALLIMAGE.BMP
Описание:

Определяет картинку в формате .bmp, которая будет отображаться в правом верхнем углу окна мастера. Этот файл должен быть расположен в исходной папке скрипта, если только для него не указан полный путь или нет префикса "compiler:" - в этом случае компилятор будет искать указанный файл в своем каталоге.

256-цветные картинки могут не отобразиться правильно в 256-цветном режиме из-за скудности палитры. Максимальный размер картинки 55х55 пикселей.

Например:
WizardSmallImageFile=mysmallimage.bmp
