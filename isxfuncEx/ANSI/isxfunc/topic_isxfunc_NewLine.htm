Pascal Scripting Extended: NewLine

Прототип:

function NewLine: TMenuItem;

Описание:
Функция создает динамически, во время выполнения программы, разделитель элементов меню в виде горизонтальной черты. Созданный разделитель может быть добавлен в массив элементов меню. Разделитель может быть передан в качестве параметра функций NewMenu, NewPopupMenu или NewSubMenu.
При динамическом создании разделителя с помощью функции NewLine необходимо позаботиться о высвобождении памяти по окончании использования разделителя. Вызов методов Delete и Remove объекта TMenuItem в данном случае не будут приводить к высвобождению памяти.
Пример:

var

Items: array[0..2] of TMenuItem;
 Item: TMenuItem;
 Key: TShortCut;

begin

Key:=ShortCut(Word('X'),[ssShift]);

Items[0]:= NewItem('Пункт1', Key, false, true, N1click, 0, 'MenuItem1');

Items[1]:= NewLine;

Items[2]:= NewItem('Пункт2', 0, false, true, N2click, 0, 'MenuItem2');

Item:= NewSubMenu('&Меню1', 0, 'Submenu1', Items, true,);
 NewMenu(Form1, 'Menu1', Item);

end;
