Inno Setup Preprocessor: EncodeVer

EncodeVer

#define EncodeVer(int Major, int Minor, int Revision = 0, int Build = -1)


Описание

Выполняет кодирование номера версии приложения, состоящего из четырех отдельных элементов (Major, Minor, Revision и Build), в целое, 32-бит число (8 бит на каждый элемент версии, таким образом каждое из значений должно быть в пределах 0...255).

Функция объявлена в файле ISPPBuiltins.iss.
