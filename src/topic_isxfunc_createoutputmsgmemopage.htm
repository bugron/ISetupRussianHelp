Pascal Scripting: CreateOutputMsgMemoPage

Прототип:

function CreateOutputMsgMemoPage(const AfterID: Integer; const ACaption, ADescription, ASubCaption: String; const AMsg: AnsiString): TOutputMsgMemoWizardPage;

Описание:

Создает страницу мастера, содержащую статический текст, многострочное поле ввода, способное отображать RTF текст, такое же поле, только для чтения. Параметр ASubCaption определяет отображаемый статический текст. AMsg определяет текст поля ввода.

Например:
var
  Page: TOutputMsgMemoWizardPage;

...

// Create the page
Page := CreateOutputMsgMemoPage(wpWelcome,
  'Information', 'Please read the following important information before continuing.',
  'When you are ready to continue with Setup, click Next.',
  'Blah blah blah.');

См. AllPagesExample.iss для другого примера.
