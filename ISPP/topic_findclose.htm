Inno Setup Preprocessor: FindClose

Прототип

void FindClose(int)

Описание

FindClose завершает последовательность FindFirst/FindNext. Параметр должен быть ненулевым дескриптором поиска, возвращенным FindFirst.

Эта функция является устаревшей с версии 1.2. ISPP автоматически освобождает ресурсы, выделенные для вызова FindFirst.
