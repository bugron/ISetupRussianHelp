[Setup]: ShowLanguageDialog

Возможные значения:
yes, no или auto

Значение по умолчанию:
yes
Описание:

При значении yes и наличии нескольких параметров в секции [Languages], в начале установки появится окно, в котором пользователь сможет поменять язык с принятого по умолчанию на другой. Для справки обратитесь к описанию секции [Languages].

При значении no окно не отобразится.

При значении auto окно появится только в том случае, когда инсталлятор не найдет идентификатор языка.
