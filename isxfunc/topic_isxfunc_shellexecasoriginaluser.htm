Pascal Scripting: ShellExecAsOriginalUser

Прототип:

function ShellExecAsOriginalUser(const Verb, Filename, Params, WorkingDir: String; const ShowCmd: Integer; const Wait: TExecWait; var ErrorCode: Integer): Boolean;

Описание:

Открывает указанный файл или выполняет другое действие, определяемое переменной Verb, используя учетные данные пользователя, запустившего инсталлятор. Для более детальной информации см. ShellExec и флаг runasoriginaluser в секции [Run].

Комментарии:

Эта функция не поддерживается во время деинсталляции.

В очень исключительных случаях неисправности (например, если процесс установки неожиданно закончился), возможно для этой функции вызвать исключение, вместо простого возвращения значения False.
