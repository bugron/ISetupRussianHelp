Pascal Scripting Extended: Sin

Прототип:

function Sin(E: Extended): Extended;

Описание:

Функция Sin - математическая функция, возвращающая синус (в радианах) числа E.

Пример:

var
 float : single;

begin
// Синус 30 градусов = 0.5
 float := Sin(PI/6); // = 180/6 = 30 градусов
 ShowMessage('Sin(PI/6) = '+FloatToStr(float));
end;
