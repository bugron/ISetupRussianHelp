[Setup]: SetupMutex

Описание:

Данная директива используется для предотвращения запуска нескольких копий инсталлятора. Указывает имена одного или более семафоров (mutex, если более одного, то разделить запятыми), которые инсталлятор проверит при запуске. Если онин из них существует, инсталлятор покажет следующее сообщение: "Обнаружен запущенный экземпляр инсталлятора. Пожалуйста, закройте все экземпляры приложения, затем нажмите «OK», чтобы продолжить, или «Отмена», чтобы выйти." Если ни онин из них не существует, инсталлятор создаст семафор(ы) и продолжит работать как обычно. Значение может содержать константы.

Чтобы указать семафор, который содержит запятую в названии, нужно экранировать запятую обратной косой чертой.

Для получения подробной информации о семафорах смотрите статью про CreateMutex в справке MS SDK.

Например:
SetupMutex=MySetupsMutexName,Global\MySetupsMutexName
