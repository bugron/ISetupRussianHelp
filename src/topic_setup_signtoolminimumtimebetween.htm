[Setup]: SignToolMinimumTimeBetween

Значение по умолчанию:

0
Описание:

При значении отличной от 0, определяет минимальной количество миллисекунд между последовательными действиями компилятора цифровой подписи. Например, при значении 5000 компилятор выполнит не более чем одно действие цифровой подписи в 5 секунд. Может быть использовано для избежания проблем с ограничением скорости (rate limiting) timestamp сервисов.
