Pascal Scripting: ComparePackedVersion

Прототип:

function ComparePackedVersion(const Version1, Version2: Int64): Integer;

Описание:

Сравнивает Version1 с Version2. Возвращает знаение меньше 0, Version1 меньше чем Version2, 0, если Version1 равен Version2 и значение больше 0, если Version1 больше Version2.
