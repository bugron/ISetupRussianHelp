[Setup]: AppCopyright

Описание:

Эта не обязательная директива определяет сообщение об авторских правах в правом нижнем углу фонового окна инсталлятора. Обратите внимание, что директива будет работать только при WindowVisible=yes.

Значение этой директивы также используется в качестве значения по умолчанию для директивы VersionInfoCopyright, если та не определена.

Например:
AppCopyright=Copyright (C) 1997-2005 My Company, Inc.
