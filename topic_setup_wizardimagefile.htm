[Setup]: WizardImageFile

Значение по умолчанию:
compiler:WIZMODERNIMAGE.BMP
Описание:

Определяет имя или имена файлов в формате .bmp, для отображения в правом верхнем углу окна мастера. Этот файл(ы) должен быть расположен в исходной папке скрипта, если не указан полный путь или путь с префиксом "compiler:", то есть компилятор будет искать указанный файл в своем каталоге.

256-цветные картинки могут не отобразиться правильно в 256-цветном режиме из-за скудности палитры. Максимальный размер картинки 55х55 пикселей.

Когда указаны несколько файлов, инсталлятор автоматически выберет тот, который наилучшим образом соответствует DPI настройкам системы. В следующей таблице приведены рекомендуемые размеры .bmp:

100%55x55
125%64x68
150%83x80
175%92x97
200%110x106
225%119x123
250%138x140
Например:
WizardSmallImageFile=mysmallimage.bmp,mysmallimage2.bmp
