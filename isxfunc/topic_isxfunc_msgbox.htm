Pascal Scripting: MsgBox

Прототип:

function MsgBox(const Text: String; const Typ: TMsgBoxType; const Buttons: Integer): Integer;

Описание:

Отображает окно сообщения. Text указывает отображаемое сообщение. Typ определяет, какую иконку нужно использовать в окне сообщения. Buttons определяет, какие кнопки будут в окне сообщения. Возвращает ID*, указывающий на кнопку, который пользователь нажал, или 0 при неудачном выполнении (чего не должно произойти, если не задан неверный параметр или системные ресурсы не исчерпаны).

Комментарии:

TMsgBoxType определяется так:

TMsgBoxType = (mbInformation, mbConfirmation, mbError, mbCriticalError);

Например:
begin
  // Display a simple message box with an OK button
  MsgBox('Hello.', mbInformation, MB_OK);

  // Ask the user a Yes/No question
  if MsgBox('Are you sure?', mbConfirmation, MB_YESNO) = IDYES then
  begin
    // user clicked Yes
  end;

  // Ask the user a Yes/No question, defaulting to No
  if MsgBox('Are you sure?', mbConfirmation, MB_YESNO or MB_DEFBUTTON2) = IDYES then
  begin
    // user clicked Yes
  end;
end;
