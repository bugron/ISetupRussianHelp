[Setup]: UserInfoPage

Возможные значения:
yes или no

Значение по умолчанию:
no
Описание:

Инсталлятор отобразит страницу для ввода информации о пользователе, в которой нужно будет указать имя пользователя, название организации и по возможности серийный номер. Полученные значения будут присвоены константам {userinfoname}, {userinfoorg} и {userinfoserial}. Эти константы можно использовать в секциях [Registry] и [Ini], чтобы сохранить их значения для дальнейшего использования.

Для появления поля серийного номера, должна присутствовать функция CheckSerial.

Директивы DefaultUserInfoName, DefaultUserInfoorg и DefaultUserInfoSerial определяют имя пользователя, название организации и серийный номер по умолчанию. Если UsePreviousUserInfo=yes (по умолчанию) и инсталлятор обнаружит, что программа уже устанавливалась на компьютер пользователя, инсталлятор будет использовать имя, название организации и серийный номер, введенные при предыдущей установке.

На тихих инсталляторах будут приняты значения по умолчанию имя пользователя, организация и значение серийного номера. Инсталлятор не будет проверять, является ли имя пользователя пустым (так как у пользователя нет возможности исправить это), однако он все же проверит серийный номер.
