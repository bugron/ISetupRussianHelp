Inno Setup Preprocessor: Предопределенные переменные

Есть несколько предопределенные переменные:

__COUNTER__ int. Автоматически увеличивает значение на единицу при каждом использовании (впоследствии).
__FILE__ str. Возвращает имя текущего файла. Пустая строка для корневого файла.
__INCLUDE__ str. Возвращает путь текущего каталога (или пути, разделенные точками с запятой) установленного через #pragma include.
__LINE__ int. Возвращает номер строки в текущем файле (не перевод), в котором она используется.
__OPT_X__ void. Определен, если заданная опция установлена через #pragma option -x+. Вместо "X" может быть любая буква от "A" до "Z." Используйте функцию Defined, чтобы узнать переменная определена или нет.
__PATHFILENAME__ str. Похож на __FILE__ но, возвращает полный путь к файлу. Пустая строка для корневого файла.
__POPT_X__ void. Определен, если заданная опция установлена через #pragma parseroption -x+. Вместо "X" может быть любая буква от "A" до "Z." Используйте функцию Defined, чтобы узнать переменная определена или нет.
__WIN32__ void. Всегда определен.
ISPP_INVOKED void. Всегда определен.
PREPROCVER int. Возвращает 32-битную закодированную версию ISPP. Самый высокий байт содержит основную версию, самый низкий байт содержит номер сборки.
WINDOWS void. Всегда определен.
UNICODE void. Всегда определен для Unicode ISPP. Используйте функцию Defined, чтобы узнать переменная определена, или нет.
CompilerPath str. Указывает на папку, где находится компилятор.
SourcePath str. Указывает на папку, где находится компилятор текущий скрипт, или на текущую папку, если скрипт еще не был сохранен.
Ver int. Возвращает 32-битную закодированную версию компилятора Inno Setup. Самый высокий байт содержит основную версию, самый низкий байт обычно содержит нуль.
