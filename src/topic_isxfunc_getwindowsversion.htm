Pascal Scripting: GetWindowsVersion

Прототип:

function GetWindowsVersion: Cardinal;

Описание:

Возвращает номер версии Windows в виде одного целого числа. Старшие 8 бит указывают основную версию; следующие 8 бит указывают номер версии, младшие 16 бит указывают номер сборки. Например, эта функция вернет $05000893 на Windows 2000, который является версией 5.0.2195.

Чтобы извлечь только основной номер версии, используйте: "GetWindowsVersion shr 24". Чтобы получить только младший номер версии, используйте: "(GetWindowsVersion shr 16) and $FF". Чтобы получить только номер сборки, используйте: "GetWindowsVersion and $FFFF".

Например:
function IsWindowsXPOrLater: Boolean;
begin
  Result := (GetWindowsVersion >= $05010000);
end;
