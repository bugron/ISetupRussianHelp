[Setup]: ExtraDiskSpaceRequired

Значение по умолчанию:
0
Описание:

Размер, требуемый для установки, подсчитывается суммированием размеров файлов секции [Files]. Если требуется увеличить это значение (оно отображается при выборе папки), укажите в директиве ExtraDiskSpaceRequired количество байт, на которые надо увеличить размер. (1048576 байт = 1 МБ)
