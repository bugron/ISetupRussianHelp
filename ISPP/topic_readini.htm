Inno Setup Preprocessor: ReadIni

Прототип

str ReadIni(str 1, str 2, str 3, str? 4)

Описание

Получает значение из INI файла. Параметром 1 должно быть имя INI файла, параметр 2 - имя раздела в INI файле, третий параметр - ключ в разделе, который нужно получить. Последний дополнительный параметр может использоваться, чтобы установить значение по умолчанию, которое будет возвращено при отказе, если он будет не указан, результатом выполнения функции будет пустая строка.
